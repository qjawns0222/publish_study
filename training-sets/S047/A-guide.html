🎯 주요 학습 포인트

  • 파일 업로드 UI

  • FileReader API로 미리보기

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="upload-area"> 드롭 영역
  • <input type="file"> (숨김)
  • 안내 텍스트와 아이콘
  • 업로드된 파일 목록 표시 영역

CSS 레이아웃

  • 드롭 영역: 점선 border, 중앙 정렬
  • Drag over: 배경색 변경, border 강조
  • 파일 목록: 파일명, 크기, 삭제 버튼
  • 미리보기 이미지 (선택사항)

스타일링 세부사항

  • JavaScript: dragover, drop 이벤트 리스너
  • FileReader로 이미지 미리보기
  • 파일 크기, 확장자 검증
  • Multiple 파일 업로드 지원

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 드롭 영역 (.drop-zone): 테두리 강조, 배경색 변경

💻 구현 코드

.drop-zone:hover {
  border-color: var(--color-primary);
  background: rgba(79, 70, 229, 0.05);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 드롭 영역 border: #D1D5DB - Gray dashed
  • Drag over 배경: #E0E7FF - Light primary
  • 파일 항목 배경: #F3F4F6 - Light
  • 삭제 버튼: #EF4444 - Red

타이포그래피

  • 안내 텍스트: 1rem, 회색
  • 파일명: 0.875rem

간격

  • 드롭 영역 패딩: 48px
  • 파일 항목 패딩: 12px

3단계: JavaScript Drag & Drop

const uploadArea = document.getElementById('uploadArea');
const fileInput = document.getElementById('fileInput');

uploadArea.addEventListener('dragover', (e) => {
  e.preventDefault();
  uploadArea.classList.add('drag-over');
});

uploadArea.addEventListener('dragleave', () => {
  uploadArea.classList.remove('drag-over');
});

uploadArea.addEventListener('drop', (e) => {
  e.preventDefault();
  uploadArea.classList.remove('drag-over');
  const files = e.dataTransfer.files;
  handleFiles(files);
});

uploadArea.addEventListener('click', () => {
  fileInput.click();
});

fileInput.addEventListener('change', (e) => {
  handleFiles(e.target.files);
});

function handleFiles(files) {
  // 파일 처리 로직
  console.log('Files uploaded:', files);
}
