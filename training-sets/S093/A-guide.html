📐 퍼블리싱 요구사항

HTML 구조

  • svg 태그
  • 각 지역별 path 또는 polygon

CSS 레이아웃

  • path hover: 색상 변경
  • 툴팁: position: absolute
  • transition: 부드러운 효과

스타일링 세부사항

  • 각 path에 클릭 이벤트
  • hover 시 툴팁 표시
  • 클릭 시 해당 지역 정보 표시
  • fill, stroke 속성 활용

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 지도 영역 (.map-region): 색상 변경, 확대

💻 구현 코드

.map-region:hover {
  fill: var(--color-primary);
  transform: scale(1.05);
  cursor: pointer;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 기본 지역: #D1D5DB - Gray
  • hover 지역: #4F46E5 - Primary
  • 선택 지역: #10B981 - Green

타이포그래피

간격

💡 단계별 구현 가이드

1단계: HTML

<div class="map-container">
  <svg id="mapSvg" viewBox="0 0 800 600">
    <path id="region1" d="M 100,100 L 200,100 L 200,200 L 100,200 Z" fill="#D1D5DB" stroke="#FFF" stroke-width="2" data-name="서울"></path>
    <path id="region2" d="M 220,100 L 320,100 L 320,200 L 220,200 Z" fill="#D1D5DB" stroke="#FFF" stroke-width="2" data-name="경기"></path>
    <!-- 더 많은 지역 -->
  </svg>
  <div class="tooltip" id="tooltip"></div>
</div>

2단계: CSS

.map-container {
  position: relative;
  max-width: 800px;
  margin: 0 auto;
}

svg path {
  cursor: pointer;
  transition: fill 0.3s;
}

svg path:hover {
  fill: #4F46E5;
}

svg path.selected {
  fill: #10B981;
}

.tooltip {
  position: absolute;
  background: rgba(0, 0, 0, 0.8);
  color: white;
  padding: 8px 16px;
  border-radius: 4px;
  font-size: 14px;
  pointer-events: none;
  display: none;
  z-index: 10;
}

3단계: JavaScript

const paths = document.querySelectorAll('svg path');
const tooltip = document.getElementById('tooltip');

paths.forEach(path => {
  path.addEventListener('mouseenter', (e) => {
    const name = e.target.dataset.name;
    tooltip.textContent = name;
    tooltip.style.display = 'block';
  });

  path.addEventListener('mousemove', (e) => {
    tooltip.style.left = e.pageX + 10 + 'px';
    tooltip.style.top = e.pageY + 10 + 'px';
  });

  path.addEventListener('mouseleave', () => {
    tooltip.style.display = 'none';
  });

  path.addEventListener('click', (e) => {
    paths.forEach(p => p.classList.remove('selected'));
    e.target.classList.add('selected');

    const name = e.target.dataset.name;
    alert(`${name} 지역이 선택되었습니다`);
  });
});
