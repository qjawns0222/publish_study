🎯 주요 학습 포인트

  • 테이블의 기본 구조
  • <thead>, <tbody>, <tr>, <th>, <td> 태그
  • 테이블 스타일링: border-collapse, 줄무늬 효과
  • hover 효과로 행 강조하기

📐 퍼블리싱 요구사항

HTML 구조

  • <table> 태그 사용
  • <thead>에 헤더 행 (이름, 이메일, 역할)
  • <tbody>에 데이터 행 5개

CSS 레이아웃

  • Table: border-collapse: collapse, 전체 너비
  • Header: 배경 Primary, 텍스트 흰색
  • Cells: 패딩 12px, 테두리 1px
  • Rows: 짝수 행 배경색 (nth-child), hover 효과

스타일링 세부사항

  • 테이블 최대 너비: 800px, 중앙 정렬
  • 짝수 행: 연한 회색 배경 (#F3F4F6)
  • Row hover: 배경색 Primary 10% 투명도

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 테이블 행 (tbody tr): 배경색 변경

💻 구현 코드

tbody tr {
  transition: 0.3s ease;
}
tbody tr:hover {
  background: rgba(79, 70, 229, 0.05);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Header 배경: #4F46E5 - Primary
  • 짝수 행 배경: #F3F4F6 - Light Gray
  • Row hover: rgba(79, 70, 229, 0.1) - Primary 10%
  • Border: #E5E7EB - Light Gray

타이포그래피

  • Table 헤더: 1rem, 중간 굵기
  • Table 셀: 1rem

간격

  • Table 최대 너비: 800px
  • Cell 패딩: 12px
  • Border: 1px solid #E5E7EB

💡 단계별 구현 가이드

1단계: HTML 구조

<table>
  <thead>
    <tr>
      <th>이름</th>
      <th>이메일</th>
      <th>역할</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>홍길동</td>
      <td>hong@email.com</td>
      <td>개발자</td>
    </tr>
    <!-- 4개 더 반복 -->
  </tbody>
</table>

2단계: Table 기본

table {
  max-width: 800px;
  width: 100%;
  margin: 0 auto;
  border-collapse: collapse;
}
th, td {
  padding: 12px;
  text-align: left;
  border: 1px solid #E5E7EB;
}

3단계: Header & 줄무늬

thead {
  background: var(--color-primary);
  color: white;
}
tbody tr:nth-child(even) {
  background: #F3F4F6;
}
tbody tr:hover {
  background: rgba(79, 70, 229, 0.1);
}

✅ 완성 체크리스트

  • □ Table이 border-collapse: collapse인가요?
  • □ Header 배경이 Primary인가요?
  • □ 짝수 행에 배경색이 있나요?
  • □ Row hover 시 배경색이 변하나요?
  • □ Cell에 패딩이 있나요?
  • □ Table이 중앙 정렬되어 있나요?
