🎯 주요 학습 포인트

  • 타임라인 UI 구조
  • 수직 연결선으로 시간 흐름 표현
  • 좌우 교차 배치 (지그재그)
  • CSS :nth-child로 교차 레이아웃

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="timeline">
  • 4-5개의 타임라인 아이템
  • 각 아이템: 날짜 + 제목 + 설명
  • 중앙 점(dot) 표시

CSS 레이아웃

  • Timeline: position: relative, 중앙 세로선
  • Item: 좌우 교차 배치
  • Dot: position: absolute, 중앙 정렬
  • :nth-child(odd): 왼쪽, even: 오른쪽

스타일링 세부사항

  • 중앙 세로선: border-left 또는 pseudo-element
  • Dot: 원형, Primary 색상
  • 카드 스타일 콘텐츠 박스
  • Hover 시 강조 효과

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 타임라인 아이템 (.timeline-item): 배경색 변경, 확대

💻 구현 코드

.timeline-item {
  transition: 0.3s ease;
}
.timeline-item:hover {
  background: rgba(79, 70, 229, 0.05);
  transform: scale(1.02);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 세로선: #E5E7EB - Light gray
  • Dot: #4F46E5 - Primary
  • 카드 배경: #FFFFFF - White
  • 날짜 텍스트: #6B7280 - Gray

타이포그래피

  • 제목: 1.25rem, 굵게
  • 날짜: 0.875rem, 회색
  • 설명: 1rem

간격

  • Item 간격: 48px
  • 카드 패딩: 20px
  • Dot 크기: 16px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="timeline">
  <div class="timeline-item">
    <div class="timeline-content">
      <span class="date">2024.01</span>
      <h3>이벤트 1</h3>
      <p>설명 내용...</p>
    </div>
  </div>
  <div class="timeline-item">
    <div class="timeline-content">
      <span class="date">2024.02</span>
      <h3>이벤트 2</h3>
      <p>설명 내용...</p>
    </div>
  </div>
</div>

2단계: CSS 레이아웃

.timeline {
  position: relative;
  padding: 40px 0;
}
.timeline::before {
  content: '';
  position: absolute;
  left: 50%;
  top: 0;
  bottom: 0;
  width: 2px;
  background: #E5E7EB;
}
.timeline-item {
  position: relative;
  margin-bottom: 48px;
}
.timeline-item::before {
  content: '';
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
  width: 16px;
  height: 16px;
  border-radius: 50%;
  background: #4F46E5;
  border: 3px solid white;
  box-shadow: 0 0 0 2px #4F46E5;
}
.timeline-item:nth-child(odd) .timeline-content {
  margin-left: auto;
  margin-right: calc(50% + 30px);
}
.timeline-item:nth-child(even) .timeline-content {
  margin-left: calc(50% + 30px);
}

  • □ 중앙에 세로선이 그려졌나요?
