📐 퍼블리싱 요구사항

HTML 구조

  • svg 태그

CSS 레이아웃

  • stroke-dasharray: 경로 길이
  • stroke-dashoffset: 애니메이션
  • @keyframes: dashoffset 0으로

스타일링 세부사항

  • path의 getTotalLength() 구하기
  • dasharray = dashoffset = 길이
  • 애니메이션: dashoffset 0으로
  • stroke-linecap: round

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • SVG 패스 (path): 그리기 애니메이션

💻 구현 코드

path {
  stroke-dasharray: 1000;
  stroke-dashoffset: 1000;
  animation: draw 3s ease forwards;
}
@keyframes draw {
  to { stroke-dashoffset: 0; }
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • stroke: #4F46E5 - Primary

타이포그래피

간격

💡 단계별 구현 가이드

1단계: HTML

<svg id="pathSvg" width="600" height="400" viewBox="0 0 600 400">
  <path id="animatedPath"
        d="M 50,200 Q 150,50 250,200 T 450,200 L 550,100"
        fill="none"
        stroke="#4F46E5"
        stroke-width="4"
        stroke-linecap="round">
  </path>
</svg>
<button onclick="animatePath()">애니메이션 시작</button>

2단계: JavaScript

const path = document.getElementById('animatedPath');
const length = path.getTotalLength();

// 초기 설정
path.style.strokeDasharray = length;
path.style.strokeDashoffset = length;

function animatePath() {
  path.style.transition = 'none';
  path.style.strokeDashoffset = length;

  setTimeout(() => {
    path.style.transition = 'stroke-dashoffset 3s ease';
    path.style.strokeDashoffset = '0';
  }, 10);
}

// 자동 시작
setTimeout(animatePath, 500);

3단계: CSS 애니메이션 (대안)

@keyframes drawPath {
  to {
    stroke-dashoffset: 0;
  }
}

#animatedPath {
  stroke-dasharray: 1000; /* 경로 길이 */
  stroke-dashoffset: 1000;
  animation: drawPath 3s ease forwards;
}
