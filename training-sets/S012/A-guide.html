🎯 주요 학습 포인트

  • 로그인 폼의 전형적인 구조
  • Password Input의 보안 처리 (type="password")
  • 폼 타이틀과 폼 요소의 시각적 계층 구조
  • "로그인 유지" 체크박스와 "비밀번호 찾기" 링크 배치

📐 퍼블리싱 요구사항

HTML 구조

  • 폼 상단에 제목 (<h2>로그인</h2>)
  • Email Input (type="email")
  • Password Input (type="password")
  • 체크박스 + 라벨 (로그인 유지)
  • 제출 버튼 + 하단 링크 (비밀번호 찾기)

CSS 레이아웃

  • Form: 최대 너비 400px, 중앙 정렬, 카드 스타일 (그림자 포함)
  • 제목: 중앙 정렬, 2rem, 굵게
  • Input: 전체 너비, 패딩 12px, 테두리 스타일
  • Button: 전체 너비, Primary 배경

스타일링 세부사항

  • 체크박스와 라벨: Flexbox로 가로 정렬
  • 비밀번호 찾기 링크: 중앙 정렬, Primary 색상, 작은 크기
  • Form 패딩: 40px
  • Input 간격: 16px

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Input 필드 (input): 테두리 강조 (focus)
  • 로그인 버튼 (.login-btn): 배경색 진하게, 위로 이동

💻 구현 코드

input:focus {
  border-color: var(--color-primary);
  box-shadow: 0 0 0 3px rgba(79, 70, 229, 0.1);
}
.login-btn:hover {
  background: #3730a3;
  transform: translateY(-2px);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Form 배경: #FFFFFF - White
  • Container 배경: #F3F4F6 - Light Gray
  • 제목: #222222 - Dark
  • Input 테두리: #E5E7EB - Light Gray
  • Button 배경: #4F46E5 - Primary
  • 링크: #4F46E5 - Primary

타이포그래피

  • 제목: 2rem, 굵게
  • Label: 0.875rem, 굵게
  • Input: 1rem
  • 비밀번호 찾기 링크: 0.875rem

간격

  • Form 패딩: 40px
  • 제목 하단 마진: 32px
  • Input 간격: 16px
  • Button 상단 마진: 24px
  • 링크 상단 마진: 16px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="login-container">
  <form class="login-form">
    <h2>로그인</h2>
    <label for="email">이메일</label>
    <input type="email" id="email" placeholder="example@email.com">

    <label for="password">비밀번호</label>
    <input type="password" id="password" placeholder="••••••••">

    <div class="checkbox-wrapper">
      <input type="checkbox" id="remember">
      <label for="remember">로그인 유지</label>
    </div>

    <button type="submit">로그인</button>
    <a href="#" class="forgot-link">비밀번호를 잊으셨나요?</a>
  </form>
</div>

2단계: Container & Form 스타일

.login-container {
  min-height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
  background: #F3F4F6;
}
.login-form {
  max-width: 400px;
  padding: 40px;
  background: white;
  border-radius: 16px;
  box-shadow: 0 4px 16px rgba(0,0,0,0.1);
}

3단계: 제목 & Input

.login-form h2 {
  text-align: center;
  margin-bottom: 32px;
}
.login-form input[type="email"],
.login-form input[type="password"] {
  width: 100%;
  margin-bottom: 16px;
  padding: 12px;
  border: 1px solid #E5E7EB;
  border-radius: 8px;
}

4단계: 체크박스 & 링크

.checkbox-wrapper {
  display: flex;
  align-items: center;
  gap: 8px;
  margin: 16px 0;
}
.forgot-link {
  display: block;
  margin-top: 16px;
  text-align: center;
  font-size: 0.875rem;
  color: var(--color-primary);
}

✅ 완성 체크리스트

  • □ Form이 화면 중앙에 위치하나요?
  • □ Form에 카드 스타일(그림자, 둥근 모서리)이 있나요?
  • □ 제목이 중앙 정렬되어 있나요?
  • □ Password Input이 type="password"인가요?
  • □ 체크박스와 라벨이 가로로 정렬되어 있나요?
  • □ 버튼이 전체 너비를 차지하나요?
  • □ 비밀번호 찾기 링크가 중앙 정렬되어 있나요?
  • □ 링크 색상이 Primary인가요?
