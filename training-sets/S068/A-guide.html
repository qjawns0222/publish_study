🎯 주요 학습 포인트

  • 햄버거 메뉴 아이콘 애니메이션
  • 3개의 선이 X 모양으로 변환
  • CSS transform으로 회전
  • JavaScript 토글 상태 관리

📐 퍼블리싱 요구사항

HTML 구조

  • 햄버거 버튼 (3개의 span 라인)
  • 클릭 시 active 클래스 토글
  • 모바일 메뉴 (선택사항)

CSS 레이아웃

  • 3개 라인: width, height, background
  • active 상태: 첫 번째/마지막 라인 45도 회전
  • 중간 라인: opacity: 0
  • transition: 0.3s ease

스타일링 세부사항

  • 첫 번째 라인: rotate(45deg) + translateY
  • 중간 라인: opacity: 0 (사라짐)
  • 마지막 라인: rotate(-45deg) + translateY
  • 클릭 시 active 토글

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 햄버거 버튼 (.hamburger): X자 변형 (클릭 시)

💻 구현 코드

.hamburger:hover {
  background: rgba(79, 70, 229, 0.1);
}
.hamburger.active .line1 {
  transform: rotate(-45deg) translate(-5px, 6px);
}
.hamburger.active .line2 {
  opacity: 0;
}
.hamburger.active .line3 {
  transform: rotate(45deg) translate(-5px, -6px);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 라인 색상: #222222 - Dark
  • active 색상: #4F46E5 - Primary
  • 버튼 배경: transparent - Transparent

타이포그래피

  • 라인 높이: 3px
  • 라인 너비: 30px

간격

  • 라인 간격: 6px
  • 버튼 패딩: 8px

💡 단계별 구현 가이드

1단계: HTML 구조

<button class="hamburger" id="hamburger">
  <span class="line"></span>
  <span class="line"></span>
  <span class="line"></span>
</button>

2단계: CSS 기본 스타일

.hamburger {
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  width: 30px;
  height: 24px;
  background: transparent;
  border: none;
  cursor: pointer;
  padding: 0;
}

.line {
  width: 100%;
  height: 3px;
  background: #222;
  transition: all 0.3s ease;
  border-radius: 2px;
}

3단계: CSS Active 상태

.hamburger.active .line:nth-child(1) {
  transform: rotate(45deg) translateY(10px);
  background: #4F46E5;
}

.hamburger.active .line:nth-child(2) {
  opacity: 0;
}

.hamburger.active .line:nth-child(3) {
  transform: rotate(-45deg) translateY(-10px);
  background: #4F46E5;
}

4단계: JavaScript 토글

const hamburger = document.getElementById('hamburger');

hamburger.addEventListener('click', () => {
  hamburger.classList.toggle('active');

  // 메뉴 토글 (선택사항)
