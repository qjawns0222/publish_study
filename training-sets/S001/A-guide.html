🎯 주요 학습 포인트

  • <header>, <main>, <footer> 시맨틱 태그의 올바른 사용법
  • Flexbox를 사용한 수직 레이아웃 (display: flex, flex-direction: column)
  • flex: 1을 활용해 Main 영역이 남는 공간을 모두 차지하게 만들기
  • Header와 Footer를 화면 상단/하단에 고정하는 방법

📐 퍼블리싱 요구사항

HTML 구조

  • <header> 태그 안에 제목과 <nav> 배치
  • <nav> 안에 3개의 링크 (홈, 소개, 연락처) 포함
  • <main> 태그로 주요 콘텐츠 영역 마크업
  • <footer> 태그로 저작권 정보 표시

CSS 레이아웃

  • Body: display: flex; flex-direction: column; min-height: 100vh;
  • Main: flex: 1; (남는 공간 모두 차지)
  • Header: Primary 색상 배경 (#4F46E5), 중앙 정렬, 24px 패딩
  • Footer: 어두운 배경 (#222), 중앙 정렬, 32px 패딩
  • Main: 최대 너비 1200px, 좌우 중앙 정렬, 48px 패딩

스타일링 세부사항

  • 네비게이션 링크: 흰색, 밑줄 없음, 좌우 16px 간격
  • CSS 변수로 색상 정의 (--color-primary, --color-dark 등)
  • Pretendard 폰트 사용, line-height: 1.6

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Header 네비게이션 링크 (header nav a): 밑줄 표시, 투명도 80%

💻 구현 코드

header nav a {
  transition: 0.3s ease;
}
header nav a:hover {
  text-decoration: underline;
  opacity: 0.8;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Header 배경: #4F46E5 - Primary - Indigo
  • Footer 배경: #222222 - Dark
  • Header/Footer 텍스트: #FFFFFF - White
  • Main 텍스트: #222222 - Dark

타이포그래피

  • Header 제목: 1.5rem, 굵게 (font-weight: bold)
  • Navigation 링크: 1rem, 보통 (font-weight: normal)
  • Main 제목: 2rem, 굵게
  • Main 본문: 1rem, line-height: 1.6
  • Footer 텍스트: 0.875rem (14px)

간격

  • Header 패딩: 24px (상하좌우)
  • Main 패딩: 48px
  • Footer 패딩: 32px (상하좌우)
  • Navigation 링크 간격: 좌우 16px 마진
  • Main 문단 간격: 상단 16px 마진

💡 단계별 구현 가이드

1단계: HTML 구조 작성

<body>
  <header>
    <h1>웹사이트 헤더</h1>
    <nav>
      <a href="#">홈</a>
      <a href="#">소개</a>
      <a href="#">연락처</a>
    </nav>
  </header>
  <main>
    <h2>메인 콘텐츠</h2>
    <p>내용...</p>
  </main>
  <footer>
    <p>© 2024 웹 퍼블리싱 훈련</p>
  </footer>
</body>

2단계: CSS 변수 정의

:root {
  --color-primary: #4F46E5;
  --color-dark: #222222;
  --color-white: #FFFFFF;
  --spacing-3: 24px;
  --spacing-4: 32px;
  --spacing-6: 48px;
}

3단계: Flexbox 레이아웃 적용

body {
  margin: 0;
  min-height: 100vh;
  display: flex;
  flex-direction: column;
}
main {
  flex: 1;
}

4단계: Header, Main, Footer 스타일링

각 섹션에 배경색, 패딩, 중앙 정렬 적용

5단계: Navigation 링크 hover 효과 추가

위의 "인터랙션 상세 요구사항" 참고

✅ 완성 체크리스트

  • □ <header>, <main>, <footer> 시맨틱 태그를 사용했나요?
  • □ Body에 display: flex; flex-direction: column; min-height: 100vh;를 적용했나요?
  • □ Main에 flex: 1;을 적용해 남는 공간을 차지하나요?
  • □ Header 배경이 Primary 색상(#4F46E5)인가요?
  • □ Footer 배경이 Dark 색상(#222)인가요?
  • □ Navigation 링크에 hover 효과(밑줄, 투명도)가 있나요?
  • □ Main 영역이 최대 너비 1200px이고 중앙 정렬되었나요?
  • □ CSS 변수를 사용해 색상과 간격을 정의했나요?
  • □ 브라우저 높이가 작아도 Footer가 하단에 붙어있나요?
