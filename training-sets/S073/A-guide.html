🎯 주요 학습 포인트

  • 디자인 그리드 오버레이
  • 격자 가이드라인 표시
  • position: fixed로 고정
  • CSS Grid 시각화

📐 퍼블리싱 요구사항

HTML 구조

  • 그리드 오버레이 컨테이너
  • 12개 또는 16개의 세로 컬럼
  • 토글 버튼 (그리드 표시/숨김)

CSS 레이아웃

  • 오버레이: position: fixed, pointer-events: none
  • 컬럼: 세로 라인, 배경색 반투명
  • Grid: display: grid, grid-template-columns: repeat(12, 1fr)
  • z-index: 9999 (최상단)

스타일링 세부사항

  • 12 컬럼 그리드 시스템 시각화
  • 토글 버튼으로 표시/숨김
  • pointer-events: none (클릭 차단 안함)
  • 반투명 배경으로 컨텐츠 가이드

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 그리드 토글 버튼 (.grid-toggle): 배경색 변경

💻 구현 코드

.grid-toggle:hover {
  background: var(--color-primary);
  color: white;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 그리드 라인: rgba(79, 70, 229, 0.1) - Primary transparent
  • 그리드 테두리: rgba(79, 70, 229, 0.3) - Primary border
  • 버튼: #4F46E5 - Primary

타이포그래피

  • 버튼 텍스트: 0.875rem, 흰색

간격

  • Gap: 24px
  • 좌우 패딩: 24px

2단계: CSS - Grid Overlay

.grid-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  pointer-events: none;
  z-index: 9999;
  opacity: 0;
  transition: opacity 0.3s ease;
}

.grid-overlay.active {
  opacity: 1;
}

.grid-container {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 24px;
  max-width: 1200px;
  margin: 0 auto;
  height: 100%;
  padding: 0 24px;
}

.grid-column {
  background: rgba(79, 70, 229, 0.1);
  border-left: 1px solid rgba(79, 70, 229, 0.3);
  border-right: 1px solid rgba(79, 70, 229, 0.3);
}

.grid-toggle {
  position: fixed;
  top: 24px;
  right: 24px;
  background: #4F46E5;
  color: white;
  border: none;
  padding: 12px 24px;
  border-radius: 8px;
  cursor: pointer;
  z-index: 10000;
  transition: 0.3s;
}

.grid-toggle:hover {
  background: #4338CA;
}

3단계: JavaScript - Toggle

const gridToggle = document.getElementById('gridToggle');
const gridOverlay = document.getElementById('gridOverlay');
let isGridVisible = false;

gridToggle.addEventListener('click', () => {
  isGridVisible = !isGridVisible;
  gridOverlay.classList.toggle('active', isGridVisible);
  gridToggle.textContent = isGridVisible ? '그리드 숨기기' : '그리드 표시';
});

// 키보드 단축키 (선택사항: G 키)
document.addEventListener('keydown', (e) => {
  if (e.key === 'g' || e.key === 'G') {
    gridToggle.click();
  }
});
