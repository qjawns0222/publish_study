HTML 구조

  • 각 단계별 입력 필드
  • 진행 바 (progress indicator)
  • 이전/다음 버튼
  • 최종 제출 버튼

CSS 레이아웃

  • 단계: 기본 숨김, 현재 단계만 표시
  • 진행 바: 현재 단계에 따라 width 변경
  • 버튼: 단계에 따라 활성화/비활성화
  • 전환 효과: 페이드 인/아웃

  • JavaScript: 현재 단계(currentStep) 추적
  • 다음 버튼: 현재 단계 검증 후 이동
  • 이전 버튼: 이전 단계로 복귀
  • 마지막 단계: 제출 버튼 표시

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="multi-step-form">
  <div class="progress-bar">
    <div class="progress" id="progress"></div>
  </div>

  <div class="step active" data-step="1">
    <h2>Step 1: 개인정보</h2>
    <input type="text" placeholder="이름" required>
    <input type="email" placeholder="이메일" required>
  </div>

  <div class="step" data-step="2">
    <h2>Step 2: 주소</h2>
    <input type="text" placeholder="주소" required>
  </div>

  <div class="buttons">
    <button id="prevBtn" disabled>이전</button>
    <button id="nextBtn">다음</button>
  </div>
</div>

2단계: JavaScript 단계 전환

let currentStep = 1;
const totalSteps = 2;
const steps = document.querySelectorAll('.step');
const progress = document.getElementById('progress');
const prevBtn = document.getElementById('prevBtn');
const nextBtn = document.getElementById('nextBtn');

function showStep(step) {
  steps.forEach((s, i) => {
    s.classList.toggle('active', i + 1 === step);
  });

  progress.style.width = `${(step / totalSteps) * 100}%`;
  prevBtn.disabled = step === 1;
  nextBtn.textContent = step === totalSteps ? '제출' : '다음';
}

nextBtn.addEventListener('click', () => {
  if (currentStep < totalSteps) {
    currentStep++;
    showStep(currentStep);
  } else {
    // 제출 로직
    alert('폼 제출!');
  }
});

prevBtn.addEventListener('click', () => {
  if (currentStep > 1) {
    currentStep--;
    showStep(currentStep);
  }
});
