🎯 주요 학습 포인트

  • 이미지 갤러리의 Grid 레이아웃
  • object-fit: cover로 이미지 비율 유지
  • aspect-ratio로 이미지 높이 자동 조정
  • Grid gap으로 이미지 간격 조정

📐 퍼블리싱 요구사항

HTML 구조

  • Container <div class="gallery">
  • 6개의 이미지 (<img> 태그 또는 배경)
  • 각 이미지를 <div class="gallery-item">로 감싸기
  • Placeholder 이미지 사용 가능

CSS 레이아웃

  • Gallery: display: grid; grid-template-columns: repeat(3, 1fr);
  • Gallery: gap: 16px;
  • Gallery Item: aspect-ratio: 1 / 1; (정사각형)
  • 이미지: width: 100%; height: 100%; object-fit: cover;

스타일링 세부사항

  • Gallery 패딩: 48px
  • Gallery Item: 둥근 모서리 12px, overflow: hidden
  • Gallery Item hover: 확대 효과 (transform: scale(1.05))
  • 반응형: 768px 이하 2열, 480px 이하 1열

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 갤러리 이미지 (.gallery-img): 확대, 밝기 조정

💻 구현 코드

.gallery-img {
  transition: 0.3s ease;
}
.gallery-img:hover {
  transform: scale(1.05);
  filter: brightness(1.1);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Gallery 배경: #F3F4F6 - Light Gray
  • Gallery Item 배경: #E5E7EB - Placeholder

타이포그래피

간격

  • Gallery 패딩: 48px
  • Grid gap: 16px
  • Gallery Item 모서리: border-radius: 12px
  • Aspect ratio: 1 / 1 (정사각형)

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="gallery">
  <div class="gallery-item">
    <img src="https://via.placeholder.com/400" alt="이미지 1">
  </div>
  <div class="gallery-item">
    <img src="https://via.placeholder.com/400" alt="이미지 2">
  </div>
  <!-- 4개 더 반복 -->
</div>

2단계: Gallery Grid

.gallery {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
  padding: 48px;
  background: #F3F4F6;
}

3단계: Gallery Item

.gallery-item {
  aspect-ratio: 1 / 1;
  border-radius: 12px;
  overflow: hidden;
  transition: 0.3s ease;
}
.gallery-item:hover {
  transform: scale(1.05);
}
.gallery-item img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

4단계: 반응형

@media (max-width: 768px) {
  .gallery {
    grid-template-columns: repeat(2, 1fr);
  }
}
@media (max-width: 480px) {
  .gallery {
    grid-template-columns: 1fr;
  }
}

✅ 완성 체크리스트

  • □ Gallery가 CSS Grid로 설정되어 있나요?
  • □ 3열 그리드가 생성되었나요?
  • □ Grid gap이 16px인가요?
  • □ 이미지가 정사각형(aspect-ratio: 1 / 1)인가요?
  • □ 이미지가 object-fit: cover로 설정되었나요?
  • □ Gallery Item hover 시 확대되나요?
  • □ 768px 이하에서 2열로 변경되나요?
  • □ 480px 이하에서 1열로 변경되나요?
