🎯 주요 학습 포인트

  • 프로필 카드 컴포넌트의 구조 (이미지 + 정보)
  • border-radius: 50%로 원형 이미지 만들기
  • Flexbox 또는 일반 블록 구조로 카드 내부 정렬
  • object-fit: cover로 이미지 비율 유지

📐 퍼블리싱 요구사항

HTML 구조

  • 카드 Container <div class="profile-card"> 생성
  • 카드 내부에 이미지(<img>), 이름(<h3>), 직책(<p>), 설명(<p>) 배치
  • 이미지는 placeholder 이미지 사용 (예: https://i.pravatar.cc/120?img=12)

CSS 레이아웃

  • Card: 최대 너비 300px, 배경 흰색, 중앙 정렬 (margin: 0 auto)
  • Card: 둥근 모서리 (border-radius: 16px), 그림자 효과
  • 이미지: 너비/높이 120px, border-radius: 50%, 중앙 정렬
  • 이미지 테두리: 4px solid #F3F4F6 (Light 색상)
  • 텍스트: 중앙 정렬 (text-align: center)

스타일링 세부사항

  • 카드 패딩: 32px
  • 카드 그림자: box-shadow: 0 4px 16px rgba(0,0,0,0.1)
  • 카드 hover 그림자: box-shadow: 0 8px 24px rgba(0,0,0,0.15)
  • 이미지와 이름 간격: 16px
  • 이미지 테두리: 4px solid #F3F4F6
  • 이름: 1.5rem, 굵게, Dark 색상
  • 직책: 1rem, Primary 색상, font-weight: 500
  • 설명: 0.875rem, 회색(#666), line-height: 1.6

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 프로필 카드 (.profile-card): 위로 이동, 그림자 강화

💻 구현 코드

.profile-card {
  transition: 0.3s ease;
}
.profile-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 24px rgba(0,0,0,0.12);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Card 배경: #FFFFFF - White
  • 이름: #222222 - Dark
  • 직책: #4F46E5 - Primary
  • 설명: #666666 - Gray

타이포그래피

  • 이름: 1.5rem, 굵게
  • 직책: 1rem, 중간 굵기 (font-weight: 500)
  • 설명: 0.875rem, line-height: 1.6

간격

  • Card 패딩: 32px
  • 이미지-이름 간격: 16px
  • 이름-직책 간격: 8px
  • 직책-설명 간격: 12px
  • 이미지 크기: 120px × 120px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="profile-card">
  <img src="https://i.pravatar.cc/120?img=12" alt="프로필">
  <h3>홍길동</h3>
  <p class="role">Frontend Developer</p>
  <p class="desc">웹 퍼블리싱과 UI 개발을 담당하고 있습니다.</p>
</div>

2단계: Card 스타일

.profile-card {
  max-width: 300px;
  margin: 0 auto;
  padding: 32px;
  background: white;
  border-radius: 16px;
  box-shadow: 0 4px 16px rgba(0,0,0,0.1);
  text-align: center;
}

3단계: 이미지 스타일

.profile-card img {
  width: 120px;
  height: 120px;
  border-radius: 50%;
  object-fit: cover;
  border: 4px solid var(--color-light);
  margin-bottom: 16px;
}

4단계: 텍스트 스타일

.profile-card h3 {
  font-size: 1.5rem;
  margin-bottom: 8px;
}
.profile-card .role {
  color: var(--color-primary);
  font-weight: 500;
  margin-bottom: 12px;
}
.profile-card .desc {
  font-size: 0.875rem;
  color: #666;
  line-height: 1.6;
}

✅ 완성 체크리스트

  • □ 카드가 최대 너비 300px이고 중앙 정렬되었나요?
  • □ 카드에 둥근 모서리(border-radius: 16px)가 있나요?
  • □ 카드에 그림자 효과(0 4px 16px rgba(0,0,0,0.1))가 있나요?
  • □ 이미지가 원형(border-radius: 50%)인가요?
  • □ 이미지 크기가 120px × 120px인가요?
  • □ 이미지에 4px 테두리(#F3F4F6)가 있나요?
  • □ 모든 텍스트가 중앙 정렬되어 있나요?
  • □ 이름이 1.5rem 굵게 표시되나요?
  • □ 직책이 Primary 색상(#4F46E5)인가요?
  • □ 설명 텍스트가 회색(#666)이고 작은 크기(0.875rem)인가요?
  • □ 카드 hover 시 위로 이동하고 그림자가 진해지나요?
