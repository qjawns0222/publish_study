🎯 주요 학습 포인트

  • Full Page Scroll (섹션별 스크롤)
  • scroll-snap 속성 사용
  • 각 섹션이 화면 전체를 차지
  • 부드러운 스크롤 스냅 효과

📐 퍼블리싱 요구사항

HTML 구조

  • Container <div class="scroll-container">
  • 5개의 섹션, 각각 전체 화면 높이
  • 각 섹션에 다른 배경색과 콘텐츠

CSS 레이아웃

  • Container: scroll-snap-type: y mandatory;
  • Container: overflow-y: scroll; height: 100vh;
  • Section: scroll-snap-align: start; height: 100vh;
  • 각 섹션: Flexbox 중앙 정렬

스타일링 세부사항

  • Container: 스크롤바 숨김 (선택사항)
  • 각 섹션: 다른 배경색 (그라디언트 사용 가능)
  • 섹션 콘텐츠: 중앙 정렬, 제목 + 설명
  • Scroll behavior: smooth

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 사이드바 링크 (.sticky-sidebar a): 배경색 변경, 좌측 바

💻 구현 코드

.sticky-sidebar a {
  transition: 0.3s ease;
}
.sticky-sidebar a:hover {
  background: rgba(79, 70, 229, 0.1);
  border-left: 3px solid var(--color-primary);
  padding-left: 20px;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Section 1: #4F46E5 - Primary
  • Section 2: #10B981 - Secondary
  • Section 3: #FF9800 - Accent
  • Section 4: #EF4444 - Red
  • Section 5: #8B5CF6 - Purple

타이포그래피

  • Section 제목: 3rem, 굵게, 흰색
  • Section 설명: 1.25rem, 흰색 80%

간격

  • Section 높이: 100vh
  • Section 패딩: 48px

💡 단계별 구현 가이드

1단계: Scroll Container

.scroll-container {
  height: 100vh;
  overflow-y: scroll;
  scroll-snap-type: y mandatory;
  scroll-behavior: smooth;
}
/* 스크롤바 숨김 (선택사항) */
.scroll-container::-webkit-scrollbar {
  display: none;
}

2단계: Section 스타일

.section {
  height: 100vh;
  scroll-snap-align: start;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: 48px;
  color: white;
  text-align: center;
}

3단계: Section 배경색

.section:nth-child(1) { background: #4F46E5; }
.section:nth-child(2) { background: #10B981; }
.section:nth-child(3) { background: #FF9800; }
.section:nth-child(4) { background: #EF4444; }
.section:nth-child(5) { background: #8B5CF6; }

✅ 완성 체크리스트

  • □ Container에 scroll-snap-type이 있나요?
  • □ Container 높이가 100vh인가요?
  • □ 각 Section에 scroll-snap-align: start가 있나요?
  • □ 스크롤 시 섹션별로 스냅되나요?
  • □ 각 Section 높이가 100vh인가요?
  • □ 콘텐츠가 섹션 중앙에 정렬되어 있나요?
