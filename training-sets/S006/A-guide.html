• Secondary: 배경 #10B981, 텍스트 흰색
• Outline: 투명 배경, 테두리 #4F46E5

🎯 주요 학습 포인트

  • 버튼 컴포넌트의 기본 구조와 변형(variant) 디자인
  • CSS 클래스를 활용한 다양한 버튼 스타일 정의
  • :hover, :active 상태 스타일링
  • 버튼 접근성: cursor: pointer와 시각적 피드백

📐 퍼블리싱 요구사항

HTML 구조

  • 3가지 버튼 타입을 각각 <button> 태그로 생성
  • Primary 버튼: <button class="btn btn-primary">
  • Secondary 버튼: <button class="btn btn-secondary">
  • Outline 버튼: <button class="btn btn-outline">

CSS 레이아웃

  • 기본 버튼(.btn): 패딩 12px 24px, 둥근 모서리 8px, 폰트 1rem
  • Primary: 배경 #4F46E5, 텍스트 흰색
  • Secondary: 배경 #10B981, 텍스트 흰색
  • Outline: 배경 투명, 테두리 2px solid #4F46E5, 텍스트 #4F46E5

스타일링 세부사항

  • 모든 버튼에 transition: 0.3s ease 적용
  • Primary hover: 배경색 진하게 (#3730a3)
  • Secondary hover: 배경색 진하게 (#059669)
  • Outline hover: 배경 Primary, 텍스트 흰색
  • 모든 버튼 active 상태: 약간 눌린 효과 (transform: scale(0.98))

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Primary 버튼 (.btn-primary): 배경색 진하게, 위로 이동
  • Secondary 버튼 (.btn-secondary): 배경색 진하게, 위로 이동
  • Outline 버튼 (.btn-outline): 배경 채우기, 텍스트 흰색

💻 구현 코드

.btn-primary:hover {
  background: #3730a3;
  transform: translateY(-2px);
}
.btn-secondary:hover {
  background: #059669;
  transform: translateY(-2px);
}
.btn-outline:hover {
  background: var(--color-primary);
  color: white;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Primary 배경: #4F46E5 - Indigo
  • Primary hover: #3730a3 - Dark Indigo
  • Secondary 배경: #10B981 - Green
  • Secondary hover: #059669 - Dark Green
  • Outline 테두리/텍스트: #4F46E5 - Indigo

타이포그래피

  • 버튼 텍스트: 1rem, 중간 굵기 (font-weight: 500)

간격

  • 버튼 패딩: 12px 24px
  • 버튼 간격: 16px (좌우 마진)
  • border-radius: 8px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="button-group">
  <button class="btn btn-primary">Primary</button>
  <button class="btn btn-secondary">Secondary</button>
  <button class="btn btn-outline">Outline</button>
</div>

2단계: 기본 버튼 스타일

.btn {
  padding: 12px 24px;
  border: none;
  border-radius: 8px;
  font-size: 1rem;
  font-weight: 500;
  cursor: pointer;
  transition: 0.3s ease;
}
.btn:active {
  transform: scale(0.98);
}

3단계: Primary 버튼

.btn-primary {
  background: var(--color-primary);
  color: white;
}
.btn-primary:hover {
  background: #3730a3;
}

4단계: Secondary & Outline

.btn-secondary {
  background: var(--color-secondary);
  color: white;
}
.btn-secondary:hover {
  background: #059669;
}

.btn-outline {
  background: transparent;
  border: 2px solid var(--color-primary);
  color: var(--color-primary);
}
.btn-outline:hover {
  background: var(--color-primary);
  color: white;
}
