🎯 주요 학습 포인트

  • 로딩 스피너 애니메이션
  • animation: rotate infinite로 무한 회전
  • border 트릭으로 원형 스피너 만들기
  • @keyframes rotate 정의

📐 퍼블리싱 요구사항

HTML 구조

  • 스피너 Container <div class="spinner">
  • 중앙 정렬된 레이아웃

CSS 레이아웃

  • Spinner: 원형 (border-radius: 50%), 크기 60px
  • Border: 4px solid, 상단만 Primary 색상
  • Animation: animation: rotate 1s linear infinite
  • Keyframes: 0도 → 360도 회전

스타일링 세부사항

  • Spinner border: 투명 + 상단만 색상
  • 화면 중앙 정렬 (Flexbox)
  • 회전 속도: 1초

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 스피너 (.spinner): 회전 애니메이션

💻 구현 코드

.spinner {
  animation: spin 1s linear infinite;
}
@keyframes spin {
  to { transform: rotate(360deg); }
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Spinner border (상단): #4F46E5 - Primary
  • Spinner border (나머지): transparent - Transparent
  • Container 배경: #F3F4F6 - Light Gray

타이포그래피

간격

  • Spinner 크기: 60px × 60px
  • Border 두께: 4px

💡 단계별 구현 가이드

1단계: Keyframes

@keyframes rotate {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

2단계: Container

.spinner-container {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background: #F3F4F6;
}

3단계: Spinner

.spinner {
  width: 60px;
  height: 60px;
  border: 4px solid transparent;
  border-top-color: var(--color-primary);
  border-radius: 50%;
  animation: rotate 1s linear infinite;
}

✅ 완성 체크리스트

  • □ Spinner가 원형인가요?
  • □ Spinner가 계속 회전하나요?
  • □ 회전 속도가 일정한가요 (linear)?
  • □ 상단 border만 색상이 있나요?
  • □ Spinner가 화면 중앙에 있나요?
  • □ animation이 무한 반복되나요 (infinite)?
