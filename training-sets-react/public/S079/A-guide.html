📐 퍼블리싱 요구사항

HTML 구조

  • 좌/우 또는 상/하 2개 패널
  • 리사이즈 핸들 (가운데)
  • 각 패널에 콘텐츠

CSS 레이아웃

  • 컨테이너: display: flex
  • 패널: 동적 width/height
  • 핸들: 드래그 커서, 시각적 구분
  • 최소/최대: min-width, max-width

스타일링 세부사항

  • mousedown: 드래그 시작
  • mousemove: 패널 크기 조정
  • mouseup: 드래그 종료
  • clientX 또는 clientY로 위치 계산

🎨 인터랙션 상세 요구사항

  • 리사이저 핸들 (.resizer): 배경색 강조

💻 구현 코드

.resizer:hover {
  background: var(--color-primary);

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 왼쪽 패널: #F9FAFB - Light gray
  • 오른쪽 패널: #FFFFFF - White
  • 핸들: #E5E7EB - Gray
  • 핸들 hover: #4F46E5 - Primary

타이포그래피

  • 패널 제목: 1.5rem, 굵게

간격

  • 핸들 너비: 8px
  • 패널 패딩: 24px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="resizable-container">
  <div class="panel left-panel" id="leftPanel">
    <h2>왼쪽 패널</h2>
    <p>크기를 조절할 수 있습니다</p>
  </div>

  <div class="resize-handle" id="resizeHandle"></div>

  <div class="panel right-panel" id="rightPanel">
    <h2>오른쪽 패널</h2>
    <p>가운데 핸들을 드래그하세요</p>
  </div>
</div>

3단계: JavaScript - Resize

const container = document.querySelector('.resizable-container');
const leftPanel = document.getElementById('leftPanel');
const rightPanel = document.getElementById('rightPanel');
const resizeHandle = document.getElementById('resizeHandle');

let isResizing = false;

resizeHandle.addEventListener('mousedown', (e) => {
  isResizing = true;
  container.classList.add('resizing');
});

document.addEventListener('mousemove', (e) => {
  if (!isResizing) return;

  const containerRect = container.getBoundingClientRect();
  const newLeftWidth = e.clientX - containerRect.left;

  // 최소/최대 너비 체크
  const minWidth = 200;
  const maxWidth = containerRect.width * 0.8;

  if (newLeftWidth >= minWidth && newLeftWidth <= maxWidth) {
    leftPanel.style.width = `${newLeftWidth}px`;
  }
});

document.addEventListener('mouseup', () => {
  if (isResizing) {
    isResizing = false;
    container.classList.remove('resizing');
  }
});
