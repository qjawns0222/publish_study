🎯 주요 학습 포인트

  • 회원가입 폼 구조
  • 다양한 input 타입 사용
  • Form validation (HTML5)
  • 약관 동의 체크박스

📐 퍼블리싱 요구사항

HTML 구조

  • <form> 태그
  • 이름, 이메일, 비밀번호, 비밀번호 확인
  • 성별 선택 (라디오)
  • 약관 동의 (체크박스)
  • 제출 버튼

CSS 레이아웃

  • 폼 컨테이너: 최대 너비 500px, 중앙 정렬
  • Input: 전체 너비, 패딩, border
  • Label: 위쪽 정렬, 폰트 굵게
  • 버튼: 전체 너비, Primary 색상

스타일링 세부사항

  • Input에 required, pattern 속성
  • Email type으로 이메일 검증
  • 비밀번호: 최소 8자 이상
  • Placeholder 텍스트 제공

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Input 필드 (input): 테두리 강조 (focus)
  • 가입 버튼 (.signup-btn): 배경색 변경, 위로 이동

💻 구현 코드

input:focus {
  border-color: var(--color-primary);
  box-shadow: 0 0 0 3px rgba(79, 70, 229, 0.1);
}
.signup-btn:hover {
  background: #3730a3;
  transform: translateY(-2px);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Input border: #D1D5DB - Gray
  • Focus border: #4F46E5 - Primary
  • 제출 버튼: #4F46E5 - Primary
  • Label 텍스트: #374151 - Dark gray

타이포그래피

  • Label: 0.875rem, 굵게
  • Input: 1rem
  • 버튼: 1rem, 굵게

간격

  • Input 패딩: 12px
  • Field 간격: 20px
  • 버튼 패딩: 14px

💡 단계별 구현 가이드

1단계: HTML 구조

<form class="signup-form">
  <div class="form-group">
    <label for="name">이름</label>
    <input type="text" id="name" required placeholder="홍길동">
  </div>
  <div class="form-group">
    <label for="email">이메일</label>
    <input type="email" id="email" required placeholder="example@email.com">
  </div>
  <div class="form-group">
    <label for="password">비밀번호</label>
    <input type="password" id="password" minlength="8" required>
  </div>
  <div class="form-group">
    <label for="confirm">비밀번호 확인</label>
    <input type="password" id="confirm" minlength="8" required>
  </div>
  <div class="form-group">
    <label>성별</label>
    <label><input type="radio" name="gender" value="male"> 남성</label>
    <label><input type="radio" name="gender" value="female"> 여성</label>
  </div>
  <div class="form-group">
    <label><input type="checkbox" required> 약관에 동의합니다</label>
  </div>
  <button type="submit">가입하기</button>
</form>

✅ 완성 체크리스트

  • □ 모든 필드에 label이 연결되었나요?
  • □ Email 필드가 이메일 형식만 받나요?
  • □ 비밀번호가 최소 8자 이상인가요?
  • □ 약관 동의가 required인가요?
  • □ Focus 시 input 테두리 색이 변하나요?
