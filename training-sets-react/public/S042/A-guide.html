🎯 주요 학습 포인트

  • 페이지네이션 UI 구조

  • active 페이지 표시
  • 이전/다음 버튼 비활성화 로직

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="pagination">
  • 이전 버튼
  • 페이지 번호 버튼들 (1, 2, 3, ...)
  • 다음 버튼
  • 총 페이지 수 표시

CSS 레이아웃

  • 버튼: 동일한 크기, 원형 또는 둥근 사각형
  • Active: Primary 배경, 흰색 텍스트
  • Disabled: 회색, cursor: not-allowed
  • Hover: 배경색 변경

스타일링 세부사항

  • JavaScript: 현재 페이지 번호 추적
  • 페이지 번호 클릭 시 active 변경
  • 첫 페이지에서 이전 버튼 비활성화
  • 마지막 페이지에서 다음 버튼 비활성화

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 페이지 번호 (.page-num): 배경색 변경, 확대
  • 이전/다음 버튼 (.page-arrow): 배경색 변경

💻 구현 코드

.page-num:hover {
  background: var(--color-primary);
  color: white;
  transform: scale(1.1);
}
.page-arrow:hover {
  background: rgba(79, 70, 229, 0.1);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Active 페이지: #4F46E5 - Primary
  • 일반 버튼: #F3F4F6 - Light
  • Hover: #E5E7EB - Light gray darker
  • Disabled: #D1D5DB - Gray

타이포그래피

  • 페이지 번호: 1rem, 중간 굵기

간격

  • 버튼 크기: 40px × 40px
  • 버튼 간격: 4px
  • Border radius: 8px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="pagination">
  <button class="page-btn" id="prevBtn"><</button>
  <button class="page-btn active">1</button>
  <button class="page-btn">2</button>
  <button class="page-btn">3</button>
  <button class="page-btn">4</button>
  <button class="page-btn">5</button>
  <button class="page-btn" id="nextBtn">></button>
</div>

2단계: CSS 스타일

.pagination {
  display: flex;
  gap: 4px;
  align-items: center;
}
.page-btn {
  width: 40px;
  height: 40px;
  border: none;
  background: #F3F4F6;
  border-radius: 8px;
  cursor: pointer;
  transition: 0.3s;
}
.page-btn.active {
  background: #4F46E5;
  color: white;
}
.page-btn:disabled {
  background: #D1D5DB;
  cursor: not-allowed;
}

3단계: JavaScript

let currentPage = 1;
const totalPages = 5;
const pageBtns = document.querySelectorAll('.page-btn:not(#prevBtn):not(#nextBtn)');
const prevBtn = document.getElementById('prevBtn');
const nextBtn = document.getElementById('nextBtn');

function updatePagination() {
  pageBtns.forEach((btn, index) => {
    btn.classList.toggle('active', index + 1 === currentPage);
  });
  prevBtn.disabled = currentPage === 1;
  nextBtn.disabled = currentPage === totalPages;
}

pageBtns.forEach((btn, index) => {
  btn.addEventListener('click', () => {
    currentPage = index + 1;
    updatePagination();
  });
});
