🎯 주요 학습 포인트

  • 캐러셀 슬라이더 구조
  • transform translateX로 슬라이드 이동

  • 인디케이터 dots 구현

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="carousel"> 컨테이너
  • <div class="carousel-track"> 슬라이드 트랙
  • 3-5개의 슬라이드 아이템 (이미지 + 캡션)
  • 이전/다음 버튼
  • 하단 인디케이터 dots

CSS 레이아웃

  • Track: display: flex, transition: transform
  • 슬라이드: flex: 0 0 100% (전체 너비)
  • 버튼: position: absolute, 좌우 배치
  • Dots: 하단 중앙 정렬

스타일링 세부사항

  • JavaScript: currentIndex 변수로 현재 슬라이드 추적
  • 버튼 클릭 시 transform: translateX 변경
  • Dots 클릭으로 특정 슬라이드 이동
  • Active dot 표시

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 좌우 화살표 버튼 (.arrow-btn): 배경색 변경, 확대
  • 인디케이터 점 (.dot): 확대, 색상 변경

💻 구현 코드

.arrow-btn:hover {
  background: rgba(79, 70, 229, 0.9);
  transform: scale(1.1);
}
.dot:hover {
  transform: scale(1.3);
  background: var(--color-primary);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 슬라이드 배경: #F3F4F6 - Light
  • 버튼 배경: rgba(0,0,0,0.5) - Semi-transparent
  • Active dot: #4F46E5 - Primary
  • Inactive dot: #D1D5DB - Gray

타이포그래피

  • 캡션: 1.25rem, 중앙 정렬

간격

  • 슬라이드 높이: 400px
  • 버튼 크기: 40px × 40px
  • Dot 크기: 12px

  </div>
  <button class="carousel-btn prev"><</button>
  <button class="carousel-btn next">></button>

2단계: CSS 레이아웃

.carousel {
  position: relative;
  overflow: hidden;
}
.carousel-track {
  display: flex;
  transition: transform 0.5s ease;
}
.slide {
  flex: 0 0 100%;
  height: 400px;
  display: flex;
  align-items: center;
  justify-content: center;
}

3단계: JavaScript 슬라이드

let currentIndex = 0;
const track = document.querySelector('.carousel-track');
const slides = document.querySelectorAll('.slide');
const dots = document.querySelectorAll('.dot');

function updateCarousel() {
  const offset = -currentIndex * 100;
  track.style.transform = `translateX(${offset}%)`;

  dots.forEach((dot, i) => {
    dot.classList.toggle('active', i === currentIndex);
  });
}

document.querySelector('.prev').addEventListener('click', () => {
  currentIndex = (currentIndex - 1 + slides.length) % slides.length;
  updateCarousel();
});

document.querySelector('.next').addEventListener('click', () => {
  currentIndex = (currentIndex + 1) % slides.length;
  updateCarousel();
});
