  • 여러 아코디언 항목 관리

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="accordion"> 컨테이너
  • 3-4개의 아코디언 아이템
  • 각 아이템: 헤더(버튼) + 콘텐츠 영역
  • 헤더에 아이콘 (▼ 또는 +) 포함

CSS 레이아웃

  • 콘텐츠: 기본 max-height: 0, overflow: hidden
  • Open 상태: max-height: 500px (또는 auto)
  • Transition: max-height 0.3s ease
  • 헤더: 클릭 가능한 버튼 스타일

스타일링 세부사항

  • JavaScript: 헤더 클릭 시 open 클래스 토글
  • 아이콘 회전 애니메이션 (▼ → ▲)
  • 다른 아코디언 닫기 옵션 (single-open)
  • ARIA 속성으로 접근성 향상

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 아코디언 헤더 (.accordion-header): 배경색 변경

💻 구현 코드

.accordion-header {
  transition: 0.3s ease;
}
.accordion-header:hover {
  background: rgba(79, 70, 229, 0.08);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 헤더 배경: #F3F4F6 - Light
  • 헤더 hover: #E5E7EB - Light gray darker
  • 콘텐츠 배경: #FFFFFF - White
  • Border: #D1D5DB - Gray

타이포그래피

  • 헤더 텍스트: 1rem, 중간 굵기
  • 콘텐츠 텍스트: 0.9375rem, line-height 1.6

간격

  • 헤더 패딩: 16px 20px
  • 콘텐츠 패딩: 16px 20px
  • 아이템 간격: 8px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="accordion">
  <div class="accordion-item">
    <button class="accordion-header">
      <span>섹션 1</span>
      <span class="icon">▼</span>
    </button>
    <div class="accordion-content">
      <p>콘텐츠 내용...</p>
    </div>
  </div>
  <!-- 추가 아이템들 -->
</div>

2단계: CSS 스타일

.accordion-content {
  max-height: 0;
  overflow: hidden;
  transition: max-height 0.3s ease;
}
.accordion-item.open .accordion-content {
  max-height: 500px;
}
.accordion-item.open .icon {
  transform: rotate(180deg);
}

3단계: JavaScript 토글

const headers = document.querySelectorAll('.accordion-header');

headers.forEach(header => {
  header.addEventListener('click', () => {
    const item = header.parentElement;
    item.classList.toggle('open');
  });
});
