좌우 분할 인터랙티브 레이아웃

고급 기술과 최적화를 고려합니다.

🎯 주요 학습 포인트

  • 시맨틱 HTML 구조
  • Flexbox 또는 Grid 레이아웃
  • 반응형 디자인
  • 요소 정렬과 배치

📐 퍼블리싱 요구사항

기본 요구사항

  • 시맨틱 태그 사용 (header, main, section, aside, footer 등)
  • CSS Flexbox 또는 Grid로 레이아웃 구성
  • 요소 간 일관된 간격 유지 (8px scale)
  • 적절한 패딩과 마진 적용

추가 요구사항

  • 모든 인터랙티브 요소에 적절한 hover 효과 적용
  • transition은 0.3s ease 기본값 사용
  • 색상은 CSS 변수로 정의하여 일관성 유지
  • 8px 간격 시스템 준수

🎨 스타일 가이드

색상 팔레트

#4F46E5
#10B981
#FF9800
#FFFFFF
#F3F4F6
#222222

타이포그래피

  • 폰트: Pretendard, -apple-system, BlinkMacSystemFont, sans-serif
  • 제목 크기: 2.5rem (large), 2rem (medium), 1.5rem (small)
  • 본문 크기: 1rem (기본), 0.875rem (작게)
  • 행간: 1.6 (본문), 1.2 (제목)

간격 시스템 (8px scale)

  • 8px, 16px, 24px, 32px, 40px, 48px
  • 요소 간 간격은 위 값들을 조합하여 사용

💡 구현 힌트

  • 먼저 HTML 구조를 시맨틱하게 작성하세요
  • CSS는 레이아웃 → 타이포그래피 → 색상 → 인터랙션 순서로 작성하세요
  • JavaScript가 필요한 경우, 이벤트 리스너를 명확히 분리하세요
  • 크롬 개발자 도구로 요소를 검사하며 조정하세요
  • 모바일 반응형을 고려하세요 (필요시 미디어 쿼리 사용)

✅ 완성 체크리스트

  • □ HTML 구조가 시맨틱하고 명확한가?
  • □ CSS 클래스명이 의미있고 일관적인가?
  • □ 레이아웃이 의도한 대로 배치되었는가?
  • □ 모든 인터랙션이 부드럽게 작동하는가?
  • □ 색상과 타이포그래피가 일관적인가?
  • □ 코드가 깔끔하고 주석이 적절한가?
