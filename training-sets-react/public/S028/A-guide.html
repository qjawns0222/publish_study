🎯 주요 학습 포인트

  • 툴팁 구현 (CSS only)
  • ::before 또는 ::after 가상 요소
  • :hover로 툴팁 표시/숨김
  • position: absolute로 위치 조정

📐 퍼블리싱 요구사항

HTML 구조

  • 툴팁이 적용될 요소 (버튼 또는 텍스트)
  • data-tooltip 속성에 툴팁 내용

CSS 레이아웃

  • 툴팁: ::after 가상 요소 사용
  • 내용: content: attr(data-tooltip)
  • 위치: position: absolute, 요소 상단
  • 초기: opacity: 0, visibility: hidden
  • Hover: opacity: 1, visibility: visible

스타일링 세부사항

  • 툴팁 배경: Dark, 텍스트 흰색
  • 툴팁 패딩: 8px 12px, 둥근 모서리
  • 작은 삼각형 화살표 추가 (::before)
  • transition으로 부드러운 fade-in

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 툴팁 트리거 (.tooltip-trigger): 색상 변경, 툴팁 표시

💻 구현 코드

.tooltip-trigger:hover {
  color: var(--color-primary);
}
.tooltip-trigger:hover .tooltip {
  opacity: 1;
  visibility: visible;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 툴팁 배경: #222222 - Dark
  • 툴팁 텍스트: #FFFFFF - White

타이포그래피

  • 툴팁: 0.875rem

간격

  • 툴팁 패딩: 8px 12px
  • 툴팁 위치: bottom: 100%, 상단 10px 간격

💡 단계별 구현 가이드

1단계: HTML

<button class="tooltip-btn" data-tooltip="이것은 툴팁입니다">
  Hover me
</button>

2단계: 버튼 & 툴팁 기본

.tooltip-btn {
  position: relative;
  padding: 12px 24px;
  background: var(--color-primary);
  color: white;
  border: none;
  border-radius: 8px;
  cursor: pointer;
}
.tooltip-btn::after {
  content: attr(data-tooltip);
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  margin-bottom: 10px;
  padding: 8px 12px;
  background: #222;
  color: white;
  font-size: 0.875rem;
  border-radius: 6px;
  white-space: nowrap;
  opacity: 0;
  visibility: hidden;
  transition: 0.3s ease;
}

3단계: Hover 표시

.tooltip-btn:hover::after {
  opacity: 1;
  visibility: visible;
}
