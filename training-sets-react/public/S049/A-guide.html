🎯 주요 학습 포인트

  • 토글 스위치 UI
  • Checkbox를 시각적으로 변환
  • CSS로 슬라이드 애니메이션
  • On/Off 상태 표현

📐 퍼블리싱 요구사항

HTML 구조

  • <input type="checkbox">
  • <label>로 감싸기
  • 스위치 배경과 토글 원
  • 3-4개의 토글 스위치 예시

CSS 레이아웃

  • Checkbox: display: none (숨김)
  • Label: 스위치 배경 (둥근 사각형)
  • ::before: 토글 원
  • Checked: 배경 Primary, 원 이동

스타일링 세부사항

  • Transition으로 부드러운 전환
  • Checked 시 배경색 변경
  • Checked 시 원이 오른쪽으로 이동
  • Cursor: pointer로 클릭 가능 표시

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 토글 스위치 (.toggle): 배경색 밝게

💻 구현 코드

.toggle:hover {
  background: rgba(79, 70, 229, 0.7);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Off 배경: #D1D5DB - Gray
  • On 배경: #4F46E5 - Primary
  • 토글 원: #FFFFFF - White

타이포그래피

  • 레이블 텍스트: 1rem

간격

  • 스위치 크기: 60px × 32px
  • 원 크기: 24px
  • 스위치 간격: 20px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="toggle-group">
  <label class="toggle">
    <input type="checkbox">
    <span class="toggle-switch"></span>
    <span class="toggle-label">알림 받기</span>
  </label>
</div>

2단계: CSS 토글 스위치

.toggle {
  display: flex;
  align-items: center;
  cursor: pointer;
}
.toggle input {
  display: none;
}
.toggle-switch {
  position: relative;
  width: 60px;
  height: 32px;
  background: #D1D5DB;
  border-radius: 16px;
  transition: 0.3s;
  margin-right: 12px;
}
.toggle-switch::before {
  content: '';
  position: absolute;
  width: 24px;
  height: 24px;
  border-radius: 50%;
  background: white;
  top: 4px;
  left: 4px;
  transition: 0.3s;
}
.toggle input:checked + .toggle-switch {
  background: #4F46E5;
}
.toggle input:checked + .toggle-switch::before {
  left: 32px;
}
