🎯 주요 학습 포인트

  • CSS Grid를 사용한 반응형 카드 레이아웃
  • grid-template-columns: repeat(3, 1fr)로 3열 그리드 생성
  • gap 속성으로 그리드 간격 조정
  • 반응형 그리드: 화면 크기에 따라 열 개수 변경하기

📐 퍼블리싱 요구사항

HTML 구조

  • Container <div> 안에 9개의 카드 배치
  • 각 카드는 <div class="card">로 마크업
  • 카드 내부에 아이콘/이미지, 제목, 설명 포함

CSS 레이아웃

  • Container: display: grid; grid-template-columns: repeat(3, 1fr); gap: 24px;
  • Card: 배경 흰색, 둥근 모서리 (border-radius: 12px), 패딩 24px
  • Card: 그림자 효과 (box-shadow: 0 2px 8px rgba(0,0,0,0.1))
  • 반응형: 768px 이하 2열, 480px 이하 1열

스타일링 세부사항

  • Container 배경: Light Gray (#F3F4F6)
  • Container 패딩: 48px
  • Card hover: 그림자 강화, 위로 2px 이동
  • 카드 제목: 1.25rem, 굵게, Dark 색상
  • 카드 설명: 1rem, 보통, 회색(#666)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 그리드 카드 (.card): 위로 이동, 그림자 강화

💻 구현 코드

.card {
  transition: 0.3s ease;
}
.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 4px 16px rgba(0,0,0,0.15);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Container 배경: #F3F4F6 - Light Gray
  • Card 배경: #FFFFFF - White
  • 카드 제목: #222222 - Dark
  • 카드 설명: #666666 - Gray

타이포그래피

  • 카드 제목: 1.25rem, 굵게
  • 카드 설명: 1rem, 보통, line-height: 1.6

간격

  • Container 패딩: 48px
  • Grid 간격 (gap): 24px
  • Card 패딩: 24px
  • 카드 내부 요소 간격: 12px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="container">
  <div class="card">
    <div class="icon">🎨</div>
    <h3>카드 제목 1</h3>
    <p>카드 설명입니다.</p>
  </div>
  <!-- 8개 더 반복 -->
</div>

2단계: Grid 레이아웃

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
  padding: 48px;
  background: #F3F4F6;
}

3단계: Card 스타일

.card {
  background: white;
  padding: 24px;
  border-radius: 12px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  transition: 0.3s ease;
}
.card:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 16px rgba(0,0,0,0.15);
}

4단계: 반응형 미디어 쿼리

@media (max-width: 768px) {
  .container {
    grid-template-columns: repeat(2, 1fr);
  }
}
@media (max-width: 480px) {
  .container {
    grid-template-columns: 1fr;
  }
}

✅ 완성 체크리스트

  • □ Container가 CSS Grid로 설정되어 있나요?
  • □ grid-template-columns: repeat(3, 1fr)로 3열 그리드가 생성되었나요?
  • □ Grid 간격(gap)이 24px인가요?
  • □ 카드가 총 9개 배치되어 있나요?
  • □ 카드에 그림자 효과가 있나요?
  • □ 카드 hover 시 위로 이동하고 그림자가 강화되나요?
  • □ 768px 이하에서 2열로 변경되나요?
  • □ 480px 이하에서 1열로 변경되나요?
