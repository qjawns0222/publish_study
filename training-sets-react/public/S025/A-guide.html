🎯 주요 학습 포인트

  • 펄스 애니메이션 (확대/축소 반복)
  • animation-direction: alternate로 왕복
  • transform: scale로 크기 변경
  • 무한 반복 + 부드러운 easing

📐 퍼블리싱 요구사항

HTML 구조

  • 펄스 효과가 적용될 원형 요소
  • 중앙 정렬

CSS 레이아웃

  • Keyframes: pulse (scale: 1 → 1.2)
  • Animation: animation: pulse 1.5s ease-in-out infinite alternate
  • 원형 요소: 크기 100px, border-radius: 50%
  • 색상: Primary 배경

스타일링 세부사항

  • alternate로 확대/축소 반복
  • ease-in-out으로 자연스러운 움직임
  • 추가 효과: opacity 변화 (선택사항)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 펄스 요소 (.pulse): 크기 변화 (애니메이션)

💻 구현 코드

.pulse {
  animation: pulse 2s ease infinite;
}
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 원형 요소 배경: #4F46E5 - Primary
  • Container 배경: #F3F4F6 - Light Gray

타이포그래피

간격

  • 원형 요소 크기: 100px × 100px

💡 단계별 구현 가이드

1단계: Keyframes

@keyframes pulse {
  from {
    transform: scale(1);
  }
  to {
    transform: scale(1.2);
  }
}

2단계: Container

.pulse-container {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background: #F3F4F6;
}

3단계: Pulse 요소

.pulse {
  width: 100px;
  height: 100px;
  background: var(--color-primary);
  border-radius: 50%;
  animation: pulse 1.5s ease-in-out infinite alternate;
}

✅ 완성 체크리스트

  • □ @keyframes pulse가 정의되어 있나요?
  • □ 원형 요소가 확대/축소를 반복하나요?
  • □ animation-direction: alternate가 있나요?
  • □ 애니메이션이 무한 반복되나요?
  • □ ease-in-out으로 부드러운가요?
  • □ 원형 요소가 화면 중앙에 있나요?
