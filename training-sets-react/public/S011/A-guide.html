🎯 주요 학습 포인트

  • 폼(Form)의 기본 구조와 접근성
  • <label>과 <input>의 올바른 연결 (for/id 속성)
  • 폼 요소의 스타일링: border, padding, focus 상태
  • :focus 상태에서 outline 대신 border 색상 변경하기

📐 퍼블리싱 요구사항

HTML 구조

  • <form> 태그로 전체 폼 감싸기
  • Label + Input 세트 3개 (이름, 이메일, 메시지)
  • 각 Input에 id 속성, Label에 for 속성으로 연결
  • 제출 버튼 <button type="submit"> 추가

CSS 레이아웃

  • Form: 최대 너비 400px, 중앙 정렬, 배경 흰색
  • Input: 전체 너비(100%), 패딩 12px, 테두리 1px solid #E5E7EB
  • Input focus: 테두리 색상 Primary, outline 제거
  • Button: Primary 배경, 전체 너비, 패딩 12px

스타일링 세부사항

  • Label: 굵게, 0.875rem, 상단 마진 16px
  • Input: 둥근 모서리 8px, transition: 0.3s ease
  • Button: 상단 마진 24px
  • 폼 요소 간격: 16px

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Input 필드 (input): 테두리 색상 변경 (focus)
  • 제출 버튼 (.submit-btn): 배경색 진하게

💻 구현 코드

input:focus {
  border-color: var(--color-primary);
  outline: none;
  box-shadow: 0 0 0 3px rgba(79, 70, 229, 0.1);
}
.submit-btn:hover {
  background: #3730a3;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Form 배경: #FFFFFF - White
  • Input 테두리: #E5E7EB - Light Gray
  • Input focus 테두리: #4F46E5 - Primary
  • Button 배경: #4F46E5 - Primary
  • Label 텍스트: #222222 - Dark

타이포그래피

  • Label: 0.875rem, 굵게
  • Input: 1rem, 보통
  • Button: 1rem, 중간 굵기 (font-weight: 500)

간격

  • Form 최대 너비: 400px
  • Input 패딩: 12px
  • Label-Input 간격: 8px (margin-bottom)
  • Input 간 간격: 16px
  • Button 상단 마진: 24px

💡 단계별 구현 가이드

1단계: HTML 구조

<form class="basic-form">
  <label for="name">이름</label>
  <input type="text" id="name" placeholder="홍길동">

  <label for="email">이메일</label>
  <input type="email" id="email" placeholder="example@email.com">

  <label for="message">메시지</label>
  <textarea id="message" rows="4"></textarea>

  <button type="submit">제출하기</button>
</form>

2단계: Form & Label 스타일

.basic-form {
  max-width: 400px;
  margin: 0 auto;
  padding: 32px;
  background: white;
}
label {
  display: block;
  margin-top: 16px;
  margin-bottom: 8px;
  font-size: 0.875rem;
  font-weight: bold;
}

3단계: Input 스타일

input, textarea {
  width: 100%;
  padding: 12px;
  border: 1px solid #E5E7EB;
  border-radius: 8px;
  font-size: 1rem;
  transition: 0.3s ease;
}
input:focus, textarea:focus {
  outline: none;
  border-color: var(--color-primary);
}

4단계: Button 스타일

button {
  width: 100%;
  margin-top: 24px;
  padding: 12px;
  background: var(--color-primary);
  color: white;
  border: none;
  border-radius: 8px;
  font-weight: 500;
  cursor: pointer;
}

✅ 완성 체크리스트

  • □ Form이 최대 너비 400px이고 중앙 정렬되었나요?
  • □ 모든 Label이 해당 Input과 for/id로 연결되었나요?
  • □ Input 전체 너비가 100%인가요?
  • □ Input 패딩이 12px인가요?
  • □ Input focus 시 테두리 색상이 Primary로 변하나요?
  • □ Input focus 시 기본 outline이 제거되었나요?
  • □ Button이 전체 너비를 차지하나요?
  • □ Label이 굵게 표시되나요?
