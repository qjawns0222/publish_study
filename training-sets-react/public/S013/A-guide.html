🎯 주요 학습 포인트

  • 검색창(Search Input)의 기본 구조
  • Input 내부에 아이콘 배치하기 (상대/절대 위치)
  • position: relative/absolute를 활용한 레이아웃
  • Input padding 조정으로 아이콘 공간 확보

📐 퍼블리싱 요구사항

HTML 구조

  • 검색창을 감싸는 Container <div class="search-box">
  • Input (type="search" 또는 type="text")
  • 돋보기 아이콘 (SVG 또는 이모지 🔍)
  • Placeholder: "검색어를 입력하세요"

CSS 레이아웃

  • Container: position: relative, 최대 너비 500px
  • Input: 전체 너비, 좌측 패딩 40px (아이콘 공간)
  • 아이콘: position: absolute; left: 12px;
  • Input: 둥근 모서리 24px (pill 모양)

스타일링 세부사항

  • Input 높이: 48px
  • Input 테두리: 1px solid #E5E7EB
  • Input focus: 테두리 Primary, 그림자 추가
  • 아이콘 색상: 회색 (#999)
  • Container 중앙 정렬

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 검색 Input (.search-input): 테두리 강조, 그림자 추가
  • 검색 버튼 (.search-btn): 배경색 변경

💻 구현 코드

.search-input:focus {
  border-color: var(--color-primary);
  box-shadow: 0 2px 8px rgba(79, 70, 229, 0.15);
}
.search-btn:hover {
  background: #3730a3;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Input 배경: #FFFFFF - White
  • Input 테두리: #E5E7EB - Light Gray
  • Input focus 테두리: #4F46E5 - Primary
  • 아이콘: #999999 - Gray
  • Placeholder: #999999 - Gray

타이포그래피

  • Input: 1rem

간격

  • Container 최대 너비: 500px
  • Input 높이: 48px
  • Input 좌측 패딩: 40px (아이콘 공간)
  • Input 우측 패딩: 16px
  • 아이콘 위치: left: 12px, top: 50% (transform: translateY(-50%))

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="search-container">
  <div class="search-box">
    <span class="search-icon">🔍</span>
    <input type="search" placeholder="검색어를 입력하세요">
  </div>
</div>

2단계: Container & Search Box

.search-container {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
.search-box {
  position: relative;
  max-width: 500px;
  width: 100%;
}

3단계: Input 스타일

.search-box input {
  width: 100%;
  height: 48px;
  padding-left: 40px;
  padding-right: 16px;
  border: 1px solid #E5E7EB;
  border-radius: 24px;
  font-size: 1rem;
  transition: 0.3s ease;
}
.search-box input:focus {
  outline: none;
  border-color: var(--color-primary);
  box-shadow: 0 0 0 3px rgba(79, 70, 229, 0.1);
}

4단계: 아이콘 배치

.search-icon {
  position: absolute;
  left: 12px;
  top: 50%;
  transform: translateY(-50%);
  font-size: 1.25rem;
  color: #999;
  pointer-events: none;
}

✅ 완성 체크리스트

  • □ Search Box가 position: relative인가요?
  • □ Input 좌측 패딩이 40px 이상인가요?
  • □ 아이콘이 position: absolute로 배치되었나요?
  • □ 아이콘이 Input 왼쪽 내부에 위치하나요?
  • □ Input이 둥근 모서리(border-radius: 24px)인가요?
  • □ Input focus 시 테두리와 그림자가 나타나나요?
  • □ 검색창이 화면 중앙에 위치하나요?
  • □ Input 높이가 48px인가요?
