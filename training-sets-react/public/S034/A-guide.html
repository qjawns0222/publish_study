🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Masonry 아이템 (.masonry-item): 확대, 그림자

💻 구현 코드

.masonry-item {
  transition: 0.3s ease;
}
.masonry-item:hover {
  transform: scale(1.02);
  box-shadow: 0 8px 24px rgba(0,0,0,0.15);
  z-index: 10;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Header 기본 배경: rgba(255, 255, 255, 0.9) - White 90%
  • Header 스크롤 배경: #FFFFFF - White
  • Section 배경들: #F3F4F6, #E5E7EB, #D1D5DB - Grays

타이포그래피

  • Header 링크: 1rem, 중간 굵기

간격

  • Header 높이: 70px
  • Header 패딩: 0 48px
  • Section 최소 높이: 100vh

💡 단계별 구현 가이드

1단계: Sticky Header

header {
  position: sticky;
  top: 0;
  z-index: 100;
  height: 70px;
  background: rgba(255, 255, 255, 0.9);
  backdrop-filter: blur(10px);
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 0 48px;
  transition: 0.3s ease;
}
header.scrolled {
  background: white;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

2단계: Scroll 감지 JS

window.addEventListener('scroll', () => {
  const header = document.querySelector('header');
  if (window.scrollY > 0) {
    header.classList.add('scrolled');
  } else {
    header.classList.remove('scrolled');
  }
});

3단계: Content 섹션

.section {
  min-height: 100vh;
  padding: 80px 48px;
  display: flex;
  align-items: center;
  justify-content: center;
}
.section:nth-child(odd) {
  background: #F3F4F6;
}
.section:nth-child(even) {
  background: #E5E7EB;
}
