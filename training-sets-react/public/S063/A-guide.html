📐 퍼블리싱 요구사항

HTML 구조

  • 좌측: 상품 이미지 (메인 + 썸네일)
  • 우측: 제품명, 가격, 설명, 옵션, 버튼
  • 옵션: 색상 선택, 사이즈 선택
  • 수량 조절 (+ -)
  • 장바구니/구매 버튼

CSS 레이아웃

  • 레이아웃: 2열 Grid (50:50)
  • 이미지: 큰 메인 + 작은 썸네일
  • 옵션: 라디오 또는 커스텀 선택
  • 버튼: 큰 CTA 스타일

스타일링 세부사항

  • JavaScript: 썸네일 클릭으로 메인 이미지 변경
  • 수량: + - 버튼으로 조절
  • 옵션: 선택 시 시각적 피드백
  • 가격: 할인가 표시

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 썸네일 이미지 (.thumbnail): 테두리 강조
  • 옵션 버튼 (.option-btn): 배경색 변경, 테두리 강조
  • 장바구니 버튼 (.cart-btn): 배경색 변경, 위로 이동

💻 구현 코드

.thumbnail:hover {
  border-color: var(--color-primary);
}
.option-btn:hover {
  background: var(--color-primary);
  color: white;
  border-color: var(--color-primary);
}
.cart-btn:hover {
  background: #3730a3;
  transform: translateY(-2px);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 장바구니 버튼: #4F46E5 - Primary
  • 구매 버튼: #10B981 - Green
  • 원가: #9CA3AF - Gray (strikethrough)
  • 할인가: #EF4444 - Red

타이포그래피

  • 제품명: 2rem, 굵게
  • 가격: 1.5rem, 굵게

  • 섹션 간격: 32px

2단계: JavaScript

let quantity = 1;

function changeImage(thumb) {
  document.getElementById('mainImage').src = thumb.src.replace('thumb', 'main');
  document.querySelectorAll('.thumb').forEach(t => t.classList.remove('active'));
  thumb.classList.add('active');
}

function changeQuantity(delta) {
  quantity = Math.max(1, quantity + delta);
  document.getElementById('quantity').textContent = quantity;
}
