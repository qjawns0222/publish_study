🎯 주요 학습 포인트

  • 커스텀 체크박스와 라디오 버튼 디자인
  • 기본 input을 숨기고 label로 스타일링하기
  • :checked 선택자를 활용한 상태 표현
  • + (인접 형제 선택자)로 체크 상태 스타일 변경

📐 퍼블리싱 요구사항

HTML 구조

  • 체크박스 3개 (과일 선택: 사과, 바나나, 오렌지)
  • 라디오 버튼 3개 (크기 선택: S, M, L)
  • 각 input에 id, label에 for 속성으로 연결
  • 라디오 버튼은 같은 name 속성 공유

CSS 레이아웃

  • 기본 input: appearance: none 또는 숨김 처리
  • Label: ::before 가상 요소로 커스텀 박스/원 생성
  • Checked 상태: 배경 Primary, 체크 표시
  • 체크박스: 사각형 (border-radius: 4px)
  • 라디오: 원형 (border-radius: 50%)

스타일링 세부사항

  • Label: 패딩 8px, cursor: pointer
  • Checkbox 크기: 20px × 20px
  • Radio 크기: 20px × 20px
  • Checked 시 중앙에 작은 체크/점 표시

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 커스텀 체크박스 (.checkbox): 테두리 강조 (hover)
  • 커스텀 라디오 (.radio): 테두리 강조 (hover)

💻 구현 코드

.checkbox:hover, .radio:hover {
  border-color: var(--color-primary);
  box-shadow: 0 0 0 3px rgba(79, 70, 229, 0.1);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 기본 테두리: #E5E7EB - Light Gray
  • Checked 배경: #4F46E5 - Primary
  • Checked 체크 표시: #FFFFFF - White
  • Label 텍스트: #222222 - Dark

타이포그래피

  • Label: 1rem, 보통

간격

  • Checkbox/Radio 크기: 20px × 20px
  • Label 패딩: 8px
  • Input-Label 간격: 8px
  • 각 옵션 간격: 12px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="form-group">
  <h3>과일 선택 (체크박스)</h3>
  <label>
    <input type="checkbox" id="apple">
    <span>사과</span>
  </label>
  <label>
    <input type="checkbox" id="banana">
    <span>바나나</span>
  </label>
</div>

<div class="form-group">
  <h3>크기 선택 (라디오)</h3>
  <label>
    <input type="radio" name="size" value="s">
    <span>S</span>
  </label>
  <label>
    <input type="radio" name="size" value="m">
    <span>M</span>
  </label>
</div>

2단계: 기본 input 숨김

input[type="checkbox"],
input[type="radio"] {
  appearance: none;
  width: 20px;
  height: 20px;
  border: 2px solid #E5E7EB;
  cursor: pointer;
  transition: 0.3s ease;
}
input[type="checkbox"] {
  border-radius: 4px;
}
input[type="radio"] {
  border-radius: 50%;
}

3단계: Checked 상태

input[type="checkbox"]:checked,
input[type="radio"]:checked {
  background: var(--color-primary);
  border-color: var(--color-primary);
}
input[type="checkbox"]:checked::after {
  content: '✓';
  color: white;
  display: block;
  text-align: center;
  line-height: 16px;
}
input[type="radio"]:checked::after {
  content: '';
  display: block;
  width: 8px;
  height: 8px;
  background: white;
  border-radius: 50%;
  margin: 4px;
}

4단계: Label 정렬

label {
  display: flex;
  align-items: center;
  gap: 8px;
  padding: 8px;
  cursor: pointer;
}

✅ 완성 체크리스트

  • □ 체크박스가 사각형 모양인가요?
  • □ 라디오 버튼이 원형 모양인가요?
  • □ Checked 상태에서 배경이 Primary 색상으로 변하나요?
  • □ 체크박스 checked 시 체크 표시가 나타나나요?
  • □ 라디오 checked 시 중앙에 작은 점이 나타나나요?
  • □ Label 클릭 시 input이 토글되나요?
  • □ 라디오 버튼들이 같은 name으로 그룹화되었나요?
  • □ Label과 input이 Flexbox로 정렬되었나요?
