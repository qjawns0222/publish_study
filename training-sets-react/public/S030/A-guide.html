🎯 주요 학습 포인트

  • 태그(Tag) 칩 디자인
  • inline-block으로 태그 배치
  • 삭제 버튼 (× 아이콘) 추가
  • Flexbox로 태그 목록 정렬

📐 퍼블리싱 요구사항

HTML 구조

  • Container <div class="tags">
  • 각 태그 <span class="tag">
  • 태그 내부에 텍스트 + 삭제 버튼 <button>×</button>

CSS 레이아웃

  • Tag: 배경 Light Gray, 둥근 모서리, 패딩 6px 12px
  • 삭제 버튼: 배경 투명, 색상 회색, hover 시 Primary
  • Container: Flexbox wrap으로 여러 줄 가능
  • 간격: gap으로 태그 간격 조정

스타일링 세부사항

  • Tag 폰트: 0.875rem
  • 삭제 버튼: 크기 20px, 둥근
  • Tag hover: 배경색 진하게

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 태그 (.tag): 배경색 변경, 확대

💻 구현 코드

.tag {
  transition: 0.3s ease;
}
.tag:hover {
  background: var(--color-primary);
  color: white;
  transform: scale(1.05);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Tag 배경: #E5E7EB - Light Gray
  • Tag 텍스트: #222222 - Dark
  • 삭제 버튼: #999999 - Gray
  • 삭제 버튼 hover: #4F46E5 - Primary

타이포그래피

  • Tag: 0.875rem

간격

  • Tag 패딩: 6px 12px
  • Tags 간격 (gap): 8px

💡 단계별 구현 가이드

1단계: HTML

<div class="tags">
  <span class="tag">
    HTML
    <button class="tag-remove">×</button>
  </span>
  <span class="tag">
    CSS
    <button class="tag-remove">×</button>
  </span>
  <span class="tag">
    JavaScript
    <button class="tag-remove">×</button>
  </span>
</div>

2단계: Tags Container

.tags {
  display: flex;
  flex-wrap: wrap;
  gap: 8px;
}

3단계: Tag 스타일

.tag {
  display: inline-flex;
  align-items: center;
  gap: 6px;
  padding: 6px 12px;
  background: #E5E7EB;
  border-radius: 16px;
  font-size: 0.875rem;
}
.tag-remove {
  width: 20px;
  height: 20px;
  background: transparent;
  border: none;
  color: #999;
  cursor: pointer;
  transition: 0.3s ease;
}
.tag-remove:hover {
  color: var(--color-primary);
}
