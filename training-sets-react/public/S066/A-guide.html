🎯 주요 학습 포인트

  • Intersection Observer API 활용
  • 스크롤 이벤트로 요소 감지
  • 순차적 페이드 인 애니메이션
  • CSS opacity + translateY 조합

📐 퍼블리싱 요구사항

HTML 구조

  • 여러 개의 콘텐츠 섹션 (4-6개)
  • 각 섹션에 fade-in 클래스 추가
  • 섹션마다 다른 delay 적용

CSS 레이아웃

  • 초기 상태: opacity: 0, translateY(30px)
  • visible 클래스: opacity: 1, translateY(0)
  • transition: 0.6s ease
  • nth-child로 delay: 순차적 등장

스타일링 세부사항

  • Intersection Observer로 viewport 진입 감지
  • 요소가 보이면 visible 클래스 추가
  • 각 요소마다 0.1s씩 delay 증가
  • threshold: 0.2 (20% 보이면 트리거)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 페이드 요소 (.fade-element): 투명도 변화 (스크롤)

💻 구현 코드

.fade-element {
  opacity: 0;
  transform: translateY(30px);
  transition: 1s ease;
}
.fade-element.visible {
  opacity: 1;
  transform: translateY(0);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 섹션 배경: #FFFFFF - White
  • 텍스트: #222222 - Dark
  • 강조색: #4F46E5 - Primary

타이포그래피

  • 제목: 2rem, 굵게
  • 설명: 1rem, line-height 1.6

간격

  • 섹션 패딩: 80px 24px
  • 섹션 간격: 60px

💡 단계별 구현 가이드

1단계: HTML 구조

<section class="fade-section">
  <h2>첫 번째 섹션</h2>
  <p>스크롤하면 나타납니다.</p>
</section>
<section class="fade-section">
  <h2>두 번째 섹션</h2>
  <p>순차적으로 등장합니다.</p>
</section>
<!-- 추가 섹션들 -->

2단계: CSS 애니메이션

.fade-section {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}

.fade-section.visible {
  opacity: 1;
  transform: translateY(0);
}

.fade-section:nth-child(1) { transition-delay: 0s; }
.fade-section:nth-child(2) { transition-delay: 0.1s; }
.fade-section:nth-child(3) { transition-delay: 0.2s; }
.fade-section:nth-child(4) { transition-delay: 0.3s; }

3단계: JavaScript - Intersection Observer

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
      observer.unobserve(entry.target); // 한 번만 실행
    }
  });
}, {
  threshold: 0.2, // 20% 보이면 트리거
  rootMargin: '0px'
});

// 모든 섹션 관찰
document.querySelectorAll('.fade-section').forEach(section => {
  observer.observe(section);
});
