🎯 주요 학습 포인트

  • 모달 팝업 구조
  • 오버레이로 배경 어둡게
  • JavaScript로 모달 열기/닫기
  • ESC 키와 오버레이 클릭으로 닫기

📐 퍼블리싱 요구사항

HTML 구조

  • 모달 트리거 버튼
  • <div class="modal-overlay"> 배경
  • <div class="modal"> 모달 컨테이너
  • 모달 내부: 헤더, 본문, 푸터 (닫기 버튼)

CSS 레이아웃

  • 오버레이: position: fixed, 전체 화면, rgba 배경
  • 모달: position: fixed, 중앙 정렬
  • 기본 상태: display: none
  • Open 상태: display: flex

스타일링 세부사항

  • JavaScript: 버튼 클릭 시 모달 표시
  • 오버레이 클릭 또는 닫기 버튼으로 닫기
  • ESC 키로 닫기
  • 모달 열릴 때 body 스크롤 방지

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 모달 열기 버튼 (.open-modal): 배경색 변경, 위로 이동
  • 닫기 버튼 (.close-btn): 회전, 색상 변경

💻 구현 코드

.open-modal:hover {
  background: #3730a3;
  transform: translateY(-2px);
}
.close-btn:hover {
  transform: rotate(90deg);
  color: var(--color-accent);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 오버레이: rgba(0,0,0,0.5) - Semi-transparent black
  • 모달 배경: #FFFFFF - White
  • Primary 버튼: #4F46E5 - Primary
  • Close 버튼: #EF4444 - Red

타이포그래피

  • 모달 제목: 1.5rem, 굵게
  • 모달 본문: 1rem

간격

  • 모달 너비: 최대 500px
  • 모달 패딩: 32px
  • Border radius: 12px

💡 단계별 구현 가이드

1단계: HTML 구조

<button id="openModal">모달 열기</button>

<div class="modal-overlay" id="modalOverlay">
  <div class="modal">
    <div class="modal-header">
      <h2>모달 제목</h2>
      <button class="close-btn">×</button>
    </div>
    <div class="modal-body">
      <p>모달 콘텐츠...</p>
    </div>
    <div class="modal-footer">
      <button class="btn-primary">확인</button>
    </div>
  </div>
</div>

2단계: CSS 스타일

.modal-overlay {
  display: none;
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0,0,0,0.5);
  align-items: center;
  justify-content: center;
  z-index: 1000;
}
.modal-overlay.open {
  display: flex;
}
.modal {
  background: white;
  padding: 32px;
  border-radius: 12px;
  max-width: 500px;
  box-shadow: 0 8px 24px rgba(0,0,0,0.2);
}

3단계: JavaScript

const openBtn = document.getElementById('openModal');
const overlay = document.getElementById('modalOverlay');
const closeBtn = document.querySelector('.close-btn');

openBtn.addEventListener('click', () => {
  overlay.classList.add('open');
  document.body.style.overflow = 'hidden';
});

closeBtn.addEventListener('click', () => {
  overlay.classList.remove('open');
  document.body.style.overflow = '';
});

overlay.addEventListener('click', (e) => {
  if (e.target === overlay) {
    overlay.classList.remove('open');
  }
});
