🎯 주요 학습 포인트

  • 타이핑 효과 (Typewriter Effect)
  • setInterval로 한 글자씩 추가
  • 커서 깜빡임 애니메이션
  • 여러 문장 순환 (선택사항)

📐 퍼블리싱 요구사항

HTML 구조

  • 텍스트를 표시할 컨테이너
  • 커서 요소 (|)
  • data-text 속성에 전체 텍스트 저장

CSS 레이아웃

  • 커서: border-right, blink 애니메이션
  • @keyframes blink: opacity 0 ↔ 1
  • 폰트: monospace 또는 코드 스타일

스타일링 세부사항

  • setInterval로 한 글자씩 추가 (100ms 간격)
  • 전체 텍스트 완성 시 clearInterval
  • 커서는 CSS animation으로 깜빡임
  • 여러 문장 순환 시 setTimeout으로 딜레이

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 타이핑 텍스트 (.typing-text): 글자별 나타나기

💻 구현 코드

.typing-text {
  border-right: 2px solid var(--color-primary);
  animation: blink 0.7s infinite;
}
@keyframes blink {
  50% { border-color: transparent; }
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 텍스트: #222222 - Dark
  • 커서: #4F46E5 - Primary
  • 배경: #F9FAFB - Light gray

타이포그래피

  • 타이핑 텍스트: 2rem, monospace
  • 커서: 2px solid, Primary

간격

  • 컨테이너 패딩: 60px 24px
  • 텍스트 높이: 80px (최소)

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="typing-container">
  <h1 class="typing-text" id="typingText"></h1>
  <span class="cursor">|</span>
</div>

2단계: CSS 커서 애니메이션

.typing-container {
  display: flex;
  align-items: center;
  justify-content: center;
  min-height: 100vh;
  background: #F9FAFB;
}

.typing-text {
  font-size: 2rem;
  font-family: 'Courier New', monospace;
  color: #222;
}

.cursor {
  font-size: 2rem;
  color: #4F46E5;
  animation: blink 0.7s infinite;
  margin-left: 4px;
}

@keyframes blink {
  0%, 50% { opacity: 1; }
  51%, 100% { opacity: 0; }
}

3단계: JavaScript - Typewriter

const text = "안녕하세요, 웹 퍼블리싱 훈련입니다!";
const typingText = document.getElementById('typingText');
let index = 0;

function typeWriter() {
  if (index < text.length) {
    typingText.textContent += text.charAt(index);
    index++;
    setTimeout(typeWriter, 100); // 100ms 간격
  }
}

typeWriter();
