🎯 주요 학습 포인트

  • transform: perspective
  • rotateX, rotateY
  • 마우스 위치에 따른 회전

📐 퍼블리싱 요구사항

HTML 구조

  • 카드 컨테이너
  • 카드 콘텐츠

CSS 레이아웃

  • perspective: 1000px
  • transform-style: preserve-3d
  • transform: rotateX, rotateY
  • transition: 부드러운 회전

스타일링 세부사항

  • 마우스 위치 계산 (중심 기준)
  • rotateY: (x - 중심) / 너비 * 각도
  • rotateX: (y - 중심) / 높이 * 각도
  • mouseleave 시 원위치

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 3D 카드 (.card-3d): 3D 회전 (마우스 위치)

💻 구현 코드

.card-3d:hover {
  transform: perspective(1000px) rotateX(var(--rotateX)) rotateY(var(--rotateY));
}
/* JavaScript로 마우스 위치에 따라 회전 각도 조정 */
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 카드 배경: linear-gradient(135deg, #667eea, #764ba2) - Gradient
  • 텍스트: #FFFFFF - White

타이포그래피

간격

  • 카드 크기: 300x400px

💡 단계별 구현 가이드

1단계: HTML

<div class="card-3d" id="card3d">
  <div class="card-content">
    <h2>3D 카드</h2>
    <p>마우스를 올려보세요</p>
  </div>
</div>

2단계: CSS

.card-3d {
  width: 300px;
  height: 400px;
  perspective: 1000px;
  cursor: pointer;
}

.card-content {
  width: 100%;
  height: 100%;
  background: linear-gradient(135deg, #667eea, #764ba2);
  border-radius: 16px;
  padding: 48px;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  color: white;
  transform-style: preserve-3d;
  transition: transform 0.1s;
  box-shadow: 0 10px 30px rgba(0,0,0,0.3);
}

.card-content h2 {
  font-size: 2rem;
  margin-bottom: 16px;
}

.card-content p {
  font-size: 1.25rem;
}

3단계: JavaScript

const card = document.getElementById('card3d');
const cardContent = card.querySelector('.card-content');

card.addEventListener('mousemove', (e) => {
  const rect = card.getBoundingClientRect();
  const x = e.clientX - rect.left;
  const y = e.clientY - rect.top;

  const centerX = rect.width / 2;
  const centerY = rect.height / 2;

  const rotateX = (y - centerY) / centerY * -10; // -10 ~ 10도
  const rotateY = (x - centerX) / centerX * 10;

  cardContent.style.transform = `rotateX(${rotateX}deg) rotateY(${rotateY}deg)`;
});

card.addEventListener('mouseleave', () => {
  cardContent.style.transform = 'rotateX(0deg) rotateY(0deg)';
});
