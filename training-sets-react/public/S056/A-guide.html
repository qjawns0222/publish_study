📐 퍼블리싱 요구사항

HTML 구조

  • 히어로 섹션 (제목, 부제, CTA)
  • 기능 섹션 (3개 특징 카드)
  • 가격 또는 추가 CTA 섹션
  • 푸터

CSS 레이아웃

  • 히어로: 전체 너비, 배경 이미지/그라디언트
  • 기능 카드: 3열 Grid
  • CTA 버튼: 크고 눈에 띄는 스타일
  • 반응형: 모바일에서 1열로 변경

스타일링 세부사항

  • 히어로: 큰 제목, 설명, 2개 버튼
  • 기능: 아이콘 + 제목 + 설명
  • Smooth scroll to section
  • Gradient background

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • CTA 버튼 (.cta-btn): 배경색 변경, 확대
  • 기능 카드 (.feature-card): 위로 이동, 그림자

💻 구현 코드

.cta-btn:hover {
  background: #3730a3;
  transform: scale(1.05);
}
.feature-card:hover {
  transform: translateY(-8px);
  box-shadow: 0 12px 28px rgba(0,0,0,0.15);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 히어로 배경: linear-gradient(135deg, #4F46E5, #10B981) - Gradient
  • Primary CTA: #FF9800 - Accent
  • Secondary CTA: #FFFFFF - White outline
  • 기능 카드: #FFFFFF - White

타이포그래피

  • 히어로 제목: 3rem, 굵게, 흰색
  • 히어로 부제: 1.25rem, 흰색 80%
  • 기능 제목: 1.5rem, 굵게

간격

  • 히어로 높이: 최소 600px
  • 섹션 패딩: 80px 0
  • 기능 카드 gap: 32px

💡 단계별 구현 가이드

1단계: 히어로 섹션

<section class="hero">
  <div class="hero-content">
    <h1>당신의 비즈니스를 성장시키세요</h1>
    <p>최고의 솔루션으로 빠르고 효율적인 성과를 만드세요</p>
    <div class="cta-buttons">
      <button class="btn-primary">시작하기</button>
      <button class="btn-secondary">더 알아보기</button>
    </div>
  </div>
</section>

2단계: 기능 섹션

<section class="features">
  <h2>주요 기능</h2>
  <div class="feature-grid">
    <div class="feature-card">
      <div class="icon">🚀</div>
      <h3>빠른 속도</h3>
      <p>최적화된 성능으로 빠른 결과를 제공합니다.</p>
    </div>
    <!-- 추가 카드들 -->
  </div>
</section>

3단계: CSS 스타일

.hero {
  min-height: 600px;
  background: linear-gradient(135deg, #4F46E5, #10B981);
  display: flex;
  align-items: center;
  justify-content: center;
  text-align: center;
  color: white;
  padding: 80px 20px;

  margin-bottom: 16px;
}

  display: grid;
  grid-template-columns: repeat(3, 1fr);

  max-width: 1200px;
  margin: 0 auto;
}
@media (max-width: 768px) {
  .feature-grid {
    grid-template-columns: 1fr;
  }
}
