🎯 주요 학습 포인트

  • 배지(Badge) 컴포넌트의 다양한 상태 표현
  • CSS 클래스를 활용한 색상 변형 (Success, Warning, Error, Info)
  • display: inline-block과 패딩으로 배지 크기 조정
  • 작은 텍스트에 적합한 타이포그래피 설정

📐 퍼블리싱 요구사항

HTML 구조

  • 4가지 배지를 각각 <span> 태그로 생성
  • Success 배지: <span class="badge badge-success">성공</span>
  • Warning 배지: <span class="badge badge-warning">경고</span>
  • Error 배지: <span class="badge badge-error">오류</span>
  • Info 배지: <span class="badge badge-info">정보</span>

CSS 레이아웃

  • 기본 배지(.badge): display: inline-block, 패딩 6px 12px
  • 기본 배지: 폰트 0.875rem, 둥근 모서리 (border-radius: 12px)
  • Success: 배경 #10B981, 텍스트 흰색
  • Warning: 배경 #FF9800, 텍스트 흰색
  • Error: 배경 #EF4444, 텍스트 흰색
  • Info: 배경 #4F46E5, 텍스트 흰색

스타일링 세부사항

  • 배지 간격: 좌우 8px 마진
  • 폰트 굵기: 500 (중간)
  • 모든 배지에 약간의 그림자 추가 (선택사항)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 배지 (.badge): 확대, 그림자 추가

💻 구현 코드

.badge {
  transition: 0.3s ease;
}
.badge:hover {
  transform: scale(1.1);
  box-shadow: 0 2px 8px rgba(0,0,0,0.15);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Success 배경: #10B981 - Green
  • Warning 배경: #FF9800 - Orange
  • Error 배경: #EF4444 - Red
  • Info 배경: #4F46E5 - Indigo
  • 모든 배지 텍스트: #FFFFFF - White

타이포그래피

  • 배지 텍스트: 0.875rem, 중간 굵기 (font-weight: 500)

간격

  • 배지 패딩: 6px 12px
  • 배지 간격: 좌우 8px 마진
  • border-radius: 12px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="badge-container">
  <span class="badge badge-success">성공</span>
  <span class="badge badge-warning">경고</span>
  <span class="badge badge-error">오류</span>
  <span class="badge badge-info">정보</span>
</div>

2단계: 기본 배지 스타일

.badge {
  display: inline-block;
  padding: 6px 12px;
  margin: 0 8px;
  font-size: 0.875rem;
  font-weight: 500;
  color: white;
  border-radius: 12px;
}

3단계: 배지 색상 변형

.badge-success {
  background: #10B981;
}
.badge-warning {
  background: #FF9800;
}
.badge-error {
  background: #EF4444;
}
.badge-info {
  background: var(--color-primary);
}
