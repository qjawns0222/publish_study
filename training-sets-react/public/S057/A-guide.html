🎯 주요 학습 포인트

  • 가격 표 (Pricing Table) 레이아웃
  • 3단 가격 플랜 구조
  • 추천 플랜 강조 표시
  • 기능 목록과 체크마크

📐 퍼블리싱 요구사항

HTML 구조

  • 3개의 가격 카드 (Basic, Pro, Enterprise)
  • 각 카드: 플랜명, 가격, 기능 목록, 버튼
  • 추천 플랜에 "인기" 배지
  • 기능에 체크/X 마크

CSS 레이아웃

  • 3열 Grid: 동일한 높이
  • 추천 플랜: 크기 확대, 그림자 강화
  • 가격: 큰 폰트, 굵게
  • 버튼: 플랜별 다른 스타일

스타일링 세부사항

  • 추천 플랜: transform: scale(1.05)
  • 기능 목록: 체크마크 아이콘
  • Hover 효과로 카드 강조
  • 반응형: 모바일에서 1열

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 가격 카드 (.price-card): 위로 이동, 그림자 강화, 테두리 강조
  • 선택 버튼 (.select-btn): 배경색 변경

💻 구현 코드

.price-card:hover {
  transform: translateY(-8px);
  box-shadow: 0 12px 32px rgba(79, 70, 229, 0.2);
  border-color: var(--color-primary);
}
.select-btn:hover {
  background: #3730a3;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Basic 버튼: #6B7280 - Gray
  • Pro 버튼 (추천): #4F46E5 - Primary
  • Enterprise 버튼: #10B981 - Secondary
  • 배지: #FF9800 - Accent

타이포그래피

  • 플랜명: 1.5rem, 굵게
  • 가격: 3rem, 굵게
  • 기능: 1rem

간격

  • 카드 패딩: 40px
  • 카드 간격: 32px
