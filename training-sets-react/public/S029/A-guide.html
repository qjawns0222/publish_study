🎯 주요 학습 포인트

  • 브레드크럼(Breadcrumb) 네비게이션
  • 구분자(/)를 CSS로 추가하기 (::after)
  • Flexbox로 가로 정렬
  • 현재 페이지 표시

📐 퍼블리싱 요구사항

HTML 구조

  • <nav class="breadcrumb">
  • 4단계 경로 (홈 > 카테고리 > 서브카테고리 > 현재 페이지)
  • 각 단계는 <a> 또는 <span>

CSS 레이아웃

  • Container: Flexbox, 가로 정렬
  • 구분자: ::after { content: "/" }
  • 마지막 항목: 구분자 없음, 색상 Dark
  • 링크: Primary 색상, hover 효과

스타일링 세부사항

  • Breadcrumb 패딩: 16px
  • 항목 간격: 8px
  • 마지막 항목: 굵게, 링크 아님

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 브레드크럼 링크 (.breadcrumb a): 색상 변경, 밑줄

💻 구현 코드

.breadcrumb a {
  transition: 0.3s ease;
}
.breadcrumb a:hover {
  color: var(--color-primary);
  text-decoration: underline;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Breadcrumb 배경: #F3F4F6 - Light Gray
  • 링크: #4F46E5 - Primary
  • 현재 페이지: #222222 - Dark
  • 구분자: #999999 - Gray

타이포그래피

  • Breadcrumb: 0.875rem
  • 현재 페이지: 0.875rem, 굵게

간격

  • Breadcrumb 패딩: 16px
  • 항목 간격: 8px

💡 단계별 구현 가이드

1단계: HTML

<nav class="breadcrumb">
  <a href="#">홈</a>
  <a href="#">카테고리</a>
  <a href="#">서브카테고리</a>
  <span>현재 페이지</span>
</nav>

2단계: Breadcrumb 기본

.breadcrumb {
  display: flex;
  align-items: center;
  gap: 8px;
  padding: 16px;
  background: #F3F4F6;
  border-radius: 8px;
}
.breadcrumb a {
  color: var(--color-primary);
  text-decoration: none;
  font-size: 0.875rem;
}
.breadcrumb a:hover {
  text-decoration: underline;
}

3단계: 구분자 & 현재

.breadcrumb a::after {
  content: '/';
  margin-left: 8px;
  color: #999;
}
.breadcrumb span {
  font-weight: bold;
  font-size: 0.875rem;
  color: #222;
}

✅ 완성 체크리스트

  • □ Breadcrumb이 Flexbox로 정렬되어 있나요?
  • □ 각 링크 뒤에 구분자(/)가 있나요?
  • □ 마지막 항목에 구분자가 없나요?
  • □ 현재 페이지가 굵게 표시되나요?
  • □ 링크 hover 시 밑줄이 나타나나요?
  • □ 항목 간격이 일정한가요?
