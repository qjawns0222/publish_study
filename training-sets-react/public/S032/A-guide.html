🎯 주요 학습 포인트

  • Holy Grail 레이아웃 (5구역 레이아웃)
  • Header, Footer, 좌우 Sidebar, Main 영역 구성
  • Grid로 복잡한 레이아웃 간단하게 구현
  • 각 영역의 역할과 콘텐츠 배치

📐 퍼블리싱 요구사항

HTML 구조

  • Container 안에 5개 영역 (header, left-sidebar, main, right-sidebar, footer)
  • 각 영역을 시맨틱 태그로 마크업
  • Main 영역에 실제 콘텐츠

CSS 레이아웃

  • Container: Grid, 3열 (200px 1fr 200px)
  • Grid areas: header/footer는 전체 너비 차지
  • 최소 높이: 100vh
  • 각 영역: 배경색, 패딩, 테두리

스타일링 세부사항

  • Header/Footer: 전체 너비, Primary 배경
  • Left Sidebar: 200px, Light Gray 배경
  • Right Sidebar: 200px, Light Gray 배경
  • Main: 남는 공간, White 배경

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 좌측 사이드바 링크 (.left-sidebar a): 배경색 변경
  • 우측 사이드바 링크 (.right-sidebar a): 배경색 변경

💻 구현 코드

.left-sidebar a:hover, .right-sidebar a:hover {
  background: rgba(79, 70, 229, 0.08);
  padding-left: 20px;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Header/Footer: #4F46E5 - Primary
  • Sidebar: #F3F4F6 - Light Gray
  • Main: #FFFFFF - White

타이포그래피

  • Header 제목: 1.5rem, 굵게, 흰색
  • Sidebar 제목: 1rem, 굵게

간격

  • Sidebar 너비: 200px
  • Header/Footer 높이: 60px
  • 패딩: 24px

💡 단계별 구현 가이드

1단계: Grid 구조

.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header header"
    "left main right"
    "footer footer footer";
  min-height: 100vh;
}

2단계: 영역 할당

header { grid-area: header; }
.left-sidebar { grid-area: left; }
main { grid-area: main; }
.right-sidebar { grid-area: right; }
footer { grid-area: footer; }

header, footer {
  background: var(--color-primary);
  color: white;
  padding: 24px;
}
.left-sidebar, .right-sidebar {
  background: #F3F4F6;
  padding: 24px;
}
main {
  padding: 48px;
}

  • □ 좌우 Sidebar가 각각 200px인가요?
  • □ Main이 중앙의 남는 공간을 차지하나요?
  • □ 전체 높이가 최소 100vh인가요?
  • □ 각 영역이 올바른 배경색을 가지나요?
