📐 퍼블리싱 요구사항

HTML 구조

  • 카드 3개 생성
  • 각 카드에 다른 애니메이션 딜레이 적용

CSS 레이아웃

  • Keyframes: fadeIn (opacity: 0 → 1, translateY: 20px → 0)
  • Animation: animation: fadeIn 0.6s ease forwards
  • Delay: 각 카드마다 0.2s씩 증가 (0s, 0.2s, 0.4s)

스타일링 세부사항

  • 카드 초기 상태: opacity: 0
  • 카드: 배경 흰색, 그림자, 둥근 모서리
  • 애니메이션 완료 후 상태 유지 (forwards)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 카드 (.fade-card): 투명도 변화 (애니메이션)

💻 구현 코드

.fade-card {
  animation: fadeIn 1s ease;
}
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Card 배경: #FFFFFF - White
  • Container 배경: #F3F4F6 - Light Gray

타이포그래피

  • 카드 제목: 1.25rem, 굵게

간격

  • Card 패딩: 24px
  • Card 간격: 16px

💡 단계별 구현 가이드

1단계: Keyframes 정의

@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

2단계: Card 기본 스타일

.card {
  padding: 24px;
  background: white;
  border-radius: 12px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  opacity: 0;
  animation: fadeIn 0.6s ease forwards;
}

3단계: Delay 적용

.card:nth-child(1) {
  animation-delay: 0s;
}
.card:nth-child(2) {
  animation-delay: 0.2s;
}
.card:nth-child(3) {
  animation-delay: 0.4s;
}
