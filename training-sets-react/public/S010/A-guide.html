🎯 주요 학습 포인트

  • 수평 네비게이션의 기본 구조
  • Flexbox로 링크들 가로 정렬하기
  • <nav> 시맨틱 태그의 올바른 사용
  • 현재 페이지 표시 (active 상태 스타일)

📐 퍼블리싱 요구사항

HTML 구조

  • <nav> 태그 안에 5개의 링크 배치 (홈, 소개, 서비스, 블로그, 연락처)
  • 현재 페이지를 나타내는 링크에 class="active" 추가
  • 각 링크는 <a href="#"> 태그 사용

CSS 레이아웃

  • Nav: display: flex; gap: 24px;
  • Nav: 배경 Dark 색상(#222), 패딩 16px 48px
  • 링크: 흰색 텍스트, 밑줄 없음
  • Active 링크: Primary 색상 또는 밑줄 표시

스타일링 세부사항

  • 링크 hover: Primary 색상으로 변경
  • 링크 폰트: 1rem, 중간 굵기 (font-weight: 500)
  • Active 링크: 하단 보더 2px solid Primary
  • transition: 0.3s ease 적용

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Navigation 메뉴 항목 (.nav-item): 하단 밑줄, 색상 변경

💻 구현 코드

.nav-item {
  transition: 0.3s ease;
  position: relative;
}
.nav-item:hover {
  color: var(--color-primary);
}
.nav-item:hover::after {
  width: 100%;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Nav 배경: #222222 - Dark
  • 링크 기본: #FFFFFF - White
  • 링크 hover/active: #4F46E5 - Primary

타이포그래피

  • Navigation 링크: 1rem, 중간 굵기 (font-weight: 500)

간격

  • Nav 패딩: 16px 48px
  • 링크 간격 (gap): 24px
  • Active 보더: 하단 2px solid Primary

💡 단계별 구현 가이드

1단계: HTML 구조

<nav class="navbar">
  <a href="#" class="active">홈</a>
  <a href="#">소개</a>
  <a href="#">서비스</a>
  <a href="#">블로그</a>
  <a href="#">연락처</a>
</nav>

2단계: Nav 스타일

.navbar {
  display: flex;
  gap: 24px;
  padding: 16px 48px;
  background: #222;
}

3단계: 링크 스타일

.navbar a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  padding-bottom: 4px;
  border-bottom: 2px solid transparent;
  transition: 0.3s ease;
}
.navbar a:hover {
  color: var(--color-primary);
}
.navbar a.active {
  color: var(--color-primary);
  border-bottom-color: var(--color-primary);
}

✅ 완성 체크리스트

  • □ Nav가 Flexbox로 설정되어 있나요?
  • □ 링크들이 가로로 정렬되어 있나요?
  • □ 링크 간격(gap)이 24px인가요?
  • □ Nav 배경이 Dark 색상(#222)인가요?
  • □ 링크 텍스트가 흰색인가요?
  • □ 링크 hover 시 Primary 색상으로 변하나요?
  • □ Active 링크에 하단 보더가 있나요?
  • □ Active 링크가 Primary 색상인가요?
