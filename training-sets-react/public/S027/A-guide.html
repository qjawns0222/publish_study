📐 퍼블리싱 요구사항

HTML 구조

  • 프로그레스 바 Container <div class="progress">
  • 내부에 채워지는 부분 <div class="progress-fill">
  • 퍼센티지 텍스트 <span>
  • 3개의 프로그레스 바 (30%, 60%, 90%)

CSS 레이아웃

  • Container: 높이 24px, 배경 Light Gray, 둥근 모서리
  • Fill: 높이 100%, 배경 Primary, width로 진행률 표시
  • 텍스트: position: absolute로 중앙 배치
  • Transition: transition: width 0.5s ease

스타일링 세부사항

  • Progress 너비: 100%, 최대 600px
  • Fill 색상: Primary
  • 텍스트: 흰색, 굵게, 가운데 정렬

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 프로그레스 바 (.progress-bar): 그림자 강화

💻 구현 코드

.progress-bar:hover {
  box-shadow: 0 2px 8px rgba(79, 70, 229, 0.3);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Container 배경: #E5E7EB - Light Gray
  • Fill 배경: #4F46E5 - Primary
  • 텍스트: #FFFFFF - White

타이포그래피

  • 퍼센티지: 0.875rem, 굵게

간격

  • Progress 높이: 24px
  • Progress 최대 너비: 600px
  • Progress 간격: 16px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="progress">
  <div class="progress-fill" style="width: 30%;">
    <span>30%</span>
  </div>
</div>
<div class="progress">
  <div class="progress-fill" style="width: 60%;">
    <span>60%</span>
  </div>
</div>
<div class="progress">
  <div class="progress-fill" style="width: 90%;">
    <span>90%</span>
  </div>
</div>

2단계: Progress Container

.progress {
  position: relative;
  max-width: 600px;
  height: 24px;
  background: #E5E7EB;
  border-radius: 12px;
  overflow: hidden;
  margin-bottom: 16px;
}

3단계: Progress Fill

.progress-fill {
  height: 100%;
  background: var(--color-primary);
  display: flex;
  align-items: center;
  justify-content: center;
  transition: width 0.5s ease;
}
.progress-fill span {
  color: white;
  font-size: 0.875rem;
  font-weight: bold;
}

✅ 완성 체크리스트

  • □ Progress Container 높이가 24px인가요?
  • □ Fill이 Container 내부를 채우나요?
  • □ 퍼센티지 텍스트가 중앙에 있나요?
  • □ Fill에 transition이 있나요?
  • □ Progress가 둥근 모서리인가요?
  • □ 3개의 프로그레스 바가 다른 진행률을 표시하나요?
