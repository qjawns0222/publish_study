🎯 주요 학습 포인트

  • 멀티 스텝 인디케이터 UI
  • 단계별 진행 상태 표시
  • 완료/현재/대기 상태 구분
  • CSS로 단계 연결선 표현

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="stepper">
  • 3-4개의 스텝 아이템
  • 각 스텝: 번호 + 라벨
  • 스텝 사이 연결선

CSS 레이아웃

  • 스텝: Flexbox로 가로 배치
  • 번호: 원형, 중앙 정렬
  • 완료: Primary 배경, 체크 아이콘
  • 현재: Primary 보더, 흰색 배경
  • 대기: 회색

스타일링 세부사항

  • 연결선: 가로 border 또는 pseudo-element
  • 완료된 스텝은 Primary 색상
  • 현재 스텝은 강조 표시
  • JavaScript로 단계 전환

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 스텝 원 (.step): 확대, 그림자

💻 구현 코드

.step:hover {
  transform: scale(1.1);
  box-shadow: 0 4px 12px rgba(79, 70, 229, 0.3);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 완료 스텝: #4F46E5 - Primary
  • 현재 스텝 보더: #4F46E5 - Primary
  • 대기 스텝: #D1D5DB - Gray
  • 연결선: #E5E7EB - Light gray

타이포그래피

  • 스텝 번호: 1.25rem, 굵게
  • 스텝 라벨: 0.875rem

간격

  • 스텝 번호 크기: 48px × 48px
  • 스텝 간격: 24px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="stepper">
  <div class="step completed">
    <div class="step-number">✓</div>
    <div class="step-label">Step 1</div>
  </div>
  <div class="step-line"></div>
  <div class="step active">
    <div class="step-number">2</div>
    <div class="step-label">Step 2</div>
  </div>
  <div class="step-line"></div>
  <div class="step">
    <div class="step-number">3</div>
    <div class="step-label">Step 3</div>
  </div>
</div>

2단계: CSS 스타일

.stepper {
  display: flex;
  align-items: center;
  gap: 24px;
}
.step {
  display: flex;
  flex-direction: column;
  align-items: center;
}
.step-number {
  width: 48px;
  height: 48px;
  border-radius: 50%;
  display: flex;
  align-items: center;
  justify-content: center;
  font-weight: bold;
  border: 2px solid #D1D5DB;
  background: white;
}
.step.completed .step-number {
  background: #4F46E5;
  color: white;
  border-color: #4F46E5;
}
.step.active .step-number {
  border-color: #4F46E5;
  color: #4F46E5;
}
.step-line {
  flex: 1;
  height: 2px;
  background: #E5E7EB;
}
