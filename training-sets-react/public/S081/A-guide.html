🎯 주요 학습 포인트

  • 트리 뷰 (Tree View)
  • 재귀적 구조 렌더링
  • 펼치기/접기 상태 관리
  • 폴더/파일 계층 구조

📐 퍼블리싱 요구사항

HTML 구조

  • 중첩된 ul/li 구조
  • 폴더와 파일 아이콘
  • 펼치기/접기 토글 버튼
  • 들여쓰기로 계층 표시

CSS 레이아웃

  • 들여쓰기: padding-left 증가
  • 아이콘: 폴더(📁), 파일(📄)
  • 토글 버튼: ▶ ▼ 회전
  • transition: 부드러운 펼침/접힘

스타일링 세부사항

  • 각 폴더에 toggle 버튼
  • 클릭 시 자식 요소 show/hide
  • max-height transition으로 애니메이션
  • 재귀적으로 중첩된 구조 지원

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 트리 노드 (.tree-node): 배경색 변경
  • 확장/축소 아이콘 (.expand-icon): 회전

💻 구현 코드

.tree-node:hover {
  background: rgba(79, 70, 229, 0.08);
}
.tree-node.expanded .expand-icon {
  transform: rotate(90deg);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 폴더: #F59E0B - Orange
  • 파일: #6B7280 - Gray
  • hover 배경: #F3F4F6 - Light gray

타이포그래피

  • 항목 텍스트: 0.875rem, 일반
  • 아이콘: 1rem

간격

  • 들여쓰기: 20px
  • 항목 패딩: 8px 12px

💡 단계별 구현 가이드

1단계: HTML 구조

<ul class="tree">
  <li class="tree-item">
    <span class="tree-toggle">▶</span>
    <span class="tree-label">📁 폴더 1</span>
    <ul class="tree-children">
      <li class="tree-item">
        <span class="tree-label">📄 파일 1.txt</span>
      </li>
      <li class="tree-item">
        <span class="tree-toggle">▶</span>
        <span class="tree-label">📁 하위 폴더</span>
        <ul class="tree-children">
          <li class="tree-item">
            <span class="tree-label">📄 파일 2.txt</span>
          </li>
        </ul>
      </li>
    </ul>
  </li>
</ul>

2단계: CSS 스타일

.tree {
  list-style: none;
  padding: 0;
  margin: 0;
}

.tree-item {
  margin: 4px 0;
}

.tree-toggle {
  display: inline-block;
  width: 16px;
  cursor: pointer;
  transition: transform 0.3s;
  user-select: none;
}

.tree-toggle.open {
  transform: rotate(90deg);
}

.tree-label {
  padding: 8px 12px;
  border-radius: 4px;
  cursor: pointer;
  transition: background 0.15s;
}

.tree-label:hover {
  background: #F3F4F6;
}

.tree-children {
  list-style: none;
  padding-left: 20px;
  max-height: 0;
  overflow: hidden;
  transition: max-height 0.3s ease;
}

.tree-children.open {
  max-height: 1000px;
}

3단계: JavaScript

document.querySelectorAll('.tree-toggle').forEach(toggle => {
  toggle.addEventListener('click', (e) => {
    e.stopPropagation();

    const item = toggle.closest('.tree-item');
    const children = item.querySelector('.tree-children');

    if (children) {
      toggle.classList.toggle('open');
      children.classList.toggle('open');
    }
  });
});
