📐 퍼블리싱 요구사항

HTML 구조

  • 카테고리 필터 버튼들 (전체, Web, Mobile, Design)
  • 포트폴리오 아이템 그리드
  • 각 아이템: 이미지, 제목, 카테고리
  • Hover 시 오버레이 정보

CSS 레이아웃

  • 그리드: 3열, auto-fit
  • 필터 버튼: Active 상태 표시
  • 아이템: 이미지, hover 오버레이
  • 숨김: display: none 또는 opacity

스타일링 세부사항

  • JavaScript: 카테고리별 필터링
  • 전체 버튼: 모든 아이템 표시
  • Hover: 어두운 오버레이 + 텍스트
  • Transition으로 부드러운 전환

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 필터 버튼 (.filter-btn): 배경색 변경, 확대
  • 포트폴리오 아이템 (.portfolio-item): 확대, 오버레이 표시

💻 구현 코드

.filter-btn:hover {
  background: var(--color-primary);
  color: white;
  transform: scale(1.05);
}
.portfolio-item:hover {
  transform: scale(1.02);
}
.portfolio-item:hover .overlay {
  opacity: 1;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Active 필터: #4F46E5 - Primary
  • 오버레이: rgba(0,0,0,0.7) - Dark semi-transparent
  • 오버레이 텍스트: #FFFFFF - White

타이포그래피

  • 필터 버튼: 1rem
  • 아이템 제목: 1.25rem, 굵게
  • 카테고리: 0.875rem

간격

  • 그리드 gap: 24px
  • 필터 버튼 간격: 12px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="portfolio">
  <div class="filters">
    <button class="filter-btn active" data-filter="all">전체</button>
    <button class="filter-btn" data-filter="web">Web</button>
    <button class="filter-btn" data-filter="mobile">Mobile</button>
    <button class="filter-btn" data-filter="design">Design</button>
  </div>

  <div class="portfolio-grid">
    <div class="portfolio-item" data-category="web">
      <img src="project1.jpg" alt="프로젝트 1">
      <div class="overlay">
        <h3>프로젝트 1</h3>
        <p>Web Design</p>
      </div>
    </div>
    <!-- 추가 아이템들 -->
  </div>
</div>

2단계: JavaScript 필터링

const filterBtns = document.querySelectorAll('.filter-btn');
const items = document.querySelectorAll('.portfolio-item');

filterBtns.forEach(btn => {
  btn.addEventListener('click', () => {
    // Active 상태 변경
    filterBtns.forEach(b => b.classList.remove('active'));
    btn.classList.add('active');

    const filter = btn.dataset.filter;

    items.forEach(item => {
      if (filter === 'all' || item.dataset.category === filter) {
        item.style.display = 'block';
      } else {
        item.style.display = 'none';
      }
    });
  });
});
