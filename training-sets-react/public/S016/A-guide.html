🎯 주요 학습 포인트

  • 히어로 섹션(Hero Section)의 구조와 역할
  • 배경 이미지 위에 텍스트 배치하기
  • background-size: cover와 background-position: center
  • 오버레이 효과로 텍스트 가독성 높이기 (rgba 배경)

📐 퍼블리싱 요구사항

HTML 구조

  • Hero 섹션 Container <section class="hero">
  • 내부에 제목(<h1>), 부제목(<p>), CTA 버튼
  • 배경은 CSS로 처리 (이미지 URL 또는 그라디언트)

CSS 레이아웃

  • Hero: 높이 min-height: 100vh, Flexbox 중앙 정렬
  • 배경: background-image: url(...), background-size: cover
  • 오버레이: ::before 가상 요소로 어두운 배경 (rgba(0,0,0,0.5))
  • 텍스트: 흰색, 중앙 정렬, z-index로 오버레이 위에 배치

스타일링 세부사항

  • 제목: 3rem, 굵게
  • 부제목: 1.25rem, line-height: 1.6
  • CTA 버튼: Primary 배경, 패딩 16px 32px
  • 오버레이: position: absolute, width/height 100%

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • CTA 버튼 (.cta-btn): 배경색 진하게, 확대

💻 구현 코드

.cta-btn {
  transition: 0.3s ease;
}
.cta-btn:hover {
  background: #3730a3;
  transform: scale(1.05);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 오버레이: rgba(0, 0, 0, 0.5) - Black 50% opacity
  • 텍스트: #FFFFFF - White
  • CTA 버튼: #FF9800 - Accent

타이포그래피

  • 제목: 3rem, 굵게
  • 부제목: 1.25rem, line-height: 1.6
  • CTA 버튼: 1rem, 중간 굵기

간격

  • Hero 높이: min-height: 100vh
  • 제목-부제목 간격: 24px
  • 부제목-버튼 간격: 32px
  • CTA 버튼 패딩: 16px 32px

💡 단계별 구현 가이드

1단계: HTML 구조

<section class="hero">
  <div class="hero-content">
    <h1>웹 퍼블리싱의 시작</h1>
    <p>누구나 배울 수 있는 HTML, CSS, JavaScript</p>
    <button class="cta-btn">시작하기</button>
  </div>
</section>

2단계: Hero 배경 & 레이아웃

.hero {
  position: relative;
  min-height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
  background-image: url('hero-bg.jpg');
  background-size: cover;
  background-position: center;
}

3단계: 오버레이

.hero::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.5);
}

4단계: 콘텐츠 스타일

.hero-content {
  position: relative;
  z-index: 1;
  text-align: center;
  color: white;
}
.hero h1 {
  font-size: 3rem;
  margin-bottom: 24px;
}
.hero p {
  font-size: 1.25rem;
  margin-bottom: 32px;
  line-height: 1.6;
}
.cta-btn {
  padding: 16px 32px;
  background: var(--color-accent);
  color: white;
  border: none;
  border-radius: 8px;
  font-weight: 500;
  cursor: pointer;
}

✅ 완성 체크리스트

  • □ Hero 섹션 높이가 100vh인가요?
  • □ 배경 이미지가 전체를 덮고 있나요 (background-size: cover)?
  • □ 오버레이가 배경 위에 있나요?
  • □ 오버레이 투명도가 50%인가요?
  • □ 텍스트가 흰색이고 중앙 정렬되었나요?
  • □ 텍스트가 오버레이 위에 표시되나요 (z-index)?
  • □ 제목 크기가 3rem인가요?
  • □ CTA 버튼이 Accent 색상인가요?
