🎯 주요 학습 포인트

  • 반응형 디자인의 기초 - 미디어 쿼리를 사용한 화면 크기별 대응
  • Flexbox를 활용한 Header 내부 요소 배치 (로고 + 네비게이션)
  • justify-content: space-between으로 양 끝 정렬하기
  • 모바일에서는 햄버거 메뉴로 전환하는 방법

📐 퍼블리싱 요구사항

HTML 구조

  • <header> 안에 로고와 <nav> 배치
  • 로고는 <h1> 또는 <div class="logo">로 마크업
  • <nav> 안에 5개의 링크 (홈, 소개, 서비스, 블로그, 연락처)
  • 모바일용 햄버거 버튼 추가 (선택사항)

CSS 레이아웃

  • Header: display: flex; justify-content: space-between; align-items: center;
  • Header: 고정 높이 70px, 좌우 패딩 48px
  • Navigation: 링크들 가로 정렬, 각 링크 간격 24px
  • 미디어 쿼리: 768px 이하에서 네비게이션 숨김 또는 세로 정렬

스타일링 세부사항

  • Header 배경: 흰색, 하단 보더 또는 그림자 추가
  • Navigation 링크 hover: Primary 색상으로 변경
  • 로고: 1.5rem, 굵게, Primary 색상
  • 모바일(768px 이하): 네비게이션이 세로로 변경 또는 햄버거 메뉴로 전환

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Navigation 링크 (.nav a): 투명도 80%

💻 구현 코드

.nav a {
  transition: 0.3s ease;
}
.nav a:hover {
  opacity: 0.8;
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Header 배경: #FFFFFF - White
  • 로고: #4F46E5 - Primary
  • 링크 기본: #222222 - Dark
  • 링크 hover: #4F46E5 - Primary
  • 보더: #E5E7EB - Light Gray

타이포그래피

  • 로고: 1.5rem, 굵게 (font-weight: bold)
  • Navigation 링크: 1rem, 중간 굵기 (font-weight: 500)

간격

  • Header 높이: 70px
  • Header 좌우 패딩: 48px
  • Navigation 링크 간격: 24px
  • Header 하단 보더: 1px solid #E5E7EB

💡 단계별 구현 가이드

1단계: HTML 구조

<header>
  <div class="logo">MyLogo</div>
  <nav>
    <a href="#">홈</a>
    <a href="#">소개</a>
    <a href="#">서비스</a>
    <a href="#">블로그</a>
    <a href="#">연락처</a>
  </nav>
</header>

2단계: Header Flexbox 배치

header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  height: 70px;
  padding: 0 48px;
  background: white;
  border-bottom: 1px solid #E5E7EB;
}

3단계: Navigation 스타일

nav {
  display: flex;
  gap: 24px;
}
nav a {
  color: #222;
  text-decoration: none;
  font-weight: 500;
  transition: 0.3s ease;
}
nav a:hover {
  color: var(--color-primary);
}

4단계: 반응형 미디어 쿼리

@media (max-width: 768px) {
  header {
    padding: 0 24px;
  }
  nav {
    flex-direction: column;
    gap: 16px;
  }
}

  • □ Navigation 링크가 가로로 정렬되어 있나요?
