🎯 주요 학습 포인트

  • 상품 카드(Product Card)의 구조
  • 이미지, 텍스트, 가격 정보의 레이아웃
  • 카드 hover 효과와 그림자
  • 가격 표시의 시각적 강조

📐 퍼블리싱 요구사항

HTML 구조

  • 카드 Container <div class="product-card">
  • 이미지 영역 <img>
  • 상품명 <h3>
  • 가격 <p class="price">
  • 구매 버튼 (선택사항)

CSS 레이아웃

  • Card: 최대 너비 300px, 배경 흰색, 둥근 모서리
  • 이미지: 전체 너비, aspect-ratio: 4 / 3
  • Card hover: 그림자 강화, 위로 이동
  • 가격: Primary 색상, 굵게, 큰 크기

스타일링 세부사항

  • Card 패딩: 16px (하단만)
  • 이미지: 상단 둥근 모서리만 적용
  • 상품명: 1.25rem, 굵게
  • 가격: 1.5rem, Primary 색상
  • Card 그림자: 0 2px 8px rgba(0,0,0,0.1)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 상품 카드 (.product-card): 위로 이동, 그림자 강화
  • 구매 버튼 (.buy-btn): 배경색 변경

💻 구현 코드

.product-card {
  transition: 0.3s ease;
}
.product-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 24px rgba(0,0,0,0.12);
}
.buy-btn:hover {
  background: var(--color-secondary);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Card 배경: #FFFFFF - White
  • 상품명: #222222 - Dark
  • 가격: #4F46E5 - Primary

타이포그래피

  • 상품명: 1.25rem, 굵게
  • 가격: 1.5rem, 굵게

간격

  • Card 최대 너비: 300px
  • Card 패딩 (하단): 16px
  • 상품명-가격 간격: 8px
  • 이미지 aspect-ratio: 4 / 3

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="product-card">
  <img src="https://via.placeholder.com/400x300" alt="상품">
  <div class="card-content">
    <h3>상품명</h3>
    <p class="price">₩29,900</p>
  </div>
</div>

2단계: Card 스타일

.product-card {
  max-width: 300px;
  background: white;
  border-radius: 12px;
  overflow: hidden;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  transition: 0.3s ease;
}
.product-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 4px 16px rgba(0,0,0,0.15);
}

3단계: 이미지

.product-card img {
  width: 100%;
  aspect-ratio: 4 / 3;
  object-fit: cover;
}

4단계: 콘텐츠

.card-content {
  padding: 16px;
}
.card-content h3 {
  font-size: 1.25rem;
  margin-bottom: 8px;
}
.price {
  font-size: 1.5rem;
  font-weight: bold;
  color: var(--color-primary);
}

✅ 완성 체크리스트

  • □ Card 최대 너비가 300px인가요?
  • □ 이미지가 4:3 비율인가요?
  • □ 이미지가 전체 너비를 차지하나요?
  • □ Card hover 시 위로 이동하나요?
  • □ Card hover 시 그림자가 강화되나요?
  • □ 가격이 Primary 색상인가요?
  • □ 가격이 1.5rem 굵게 표시되나요?
  • □ Card에 둥근 모서리가 있나요?
