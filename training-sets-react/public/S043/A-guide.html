🎯 주요 학습 포인트

  • 3D 카드 플립 효과
  • transform rotateY 사용
  • backface-visibility로 뒷면 숨김
  • Hover 시 카드 뒤집기

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="card-flip"> 컨테이너
  • <div class="card-inner"> 회전 요소
  • <div class="card-front"> 앞면
  • <div class="card-back"> 뒷면

CSS 레이아웃

  • Inner: transform-style: preserve-3d
  • Front/Back: position: absolute, backface-visibility: hidden
  • Back: transform: rotateY(180deg)
  • Hover: Inner transform: rotateY(180deg)

스타일링 세부사항

  • Transition: transform 0.6s ease
  • 앞면: 이미지 + 제목
  • 뒷면: 설명 텍스트
  • Perspective 효과로 3D 깊이 표현

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 플립 카드 (.flip-card): 180도 회전 (3D)

💻 구현 코드

.flip-card:hover .flip-card-inner {
  transform: rotateY(180deg);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 앞면 배경: #4F46E5 - Primary
  • 뒷면 배경: #10B981 - Secondary
  • 텍스트: #FFFFFF - White

타이포그래피

  • 앞면 제목: 1.5rem, 굵게
  • 뒷면 설명: 1rem

간격

  • 카드 크기: 300px × 400px
  • 카드 패딩: 32px
  • Border radius: 12px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="card-flip">
  <div class="card-inner">
    <div class="card-front">
      <h3>앞면</h3>
      <p>Hover me!</p>
    </div>
    <div class="card-back">
      <h3>뒷면</h3>
      <p>뒷면 내용입니다.</p>
    </div>
  </div>
</div>

2단계: CSS 3D Transform

.card-flip {
  perspective: 1000px;
  width: 300px;
  height: 400px;
}
.card-inner {
  position: relative;
  width: 100%;
  height: 100%;
  transform-style: preserve-3d;
  transition: transform 0.6s;
}
.card-flip:hover .card-inner {
  transform: rotateY(180deg);
}
.card-front, .card-back {
  position: absolute;
  width: 100%;
  height: 100%;
  backface-visibility: hidden;
  border-radius: 12px;
  padding: 32px;
}
.card-front {
  background: #4F46E5;
}
.card-back {
  background: #10B981;
  transform: rotateY(180deg);
}
