🎯 주요 학습 포인트

  • 탭 UI 구조와 상태 관리
  • JavaScript로 탭 전환 구현
  • active 클래스로 선택된 탭 표시
  • 탭 콘텐츠의 show/hide 로직

📐 퍼블리싱 요구사항

HTML 구조

  • <div class="tabs"> 컨테이너
  • 탭 버튼 리스트 <div class="tab-buttons">
  • 3개의 탭 버튼 (Tab 1, Tab 2, Tab 3)
  • 탭 콘텐츠 컨테이너 <div class="tab-contents">
  • 각 탭에 대응하는 콘텐츠 패널 3개

CSS 레이아웃

  • 탭 버튼: 인라인 배치, 하단 보더로 구분
  • Active 탭: Primary 색상 하단 보더 (3px)
  • 콘텐츠: 기본적으로 숨김 (display: none)
  • Active 콘텐츠: display: block
  • 탭 전환 시 부드러운 페이드 효과

스타일링 세부사항

  • JavaScript: 각 탭 버튼에 click 이벤트 리스너
  • 클릭 시 모든 active 클래스 제거 후 선택된 탭에 추가
  • 해당 index의 콘텐츠만 표시
  • 탭 버튼 hover 시 배경색 변경

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 탭 버튼 (.tab-btn): 배경색 변경, 하단 밑줄

💻 구현 코드

.tab-btn {
  transition: 0.3s ease;
}
.tab-btn:hover {
  background: rgba(79, 70, 229, 0.1);
}
.tab-btn.active {
  border-bottom: 3px solid var(--color-primary);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 탭 버튼 배경: #F3F4F6 - Light gray
  • Active 탭 보더: #4F46E5 - Primary
  • 콘텐츠 배경: #FFFFFF - White
  • Hover 배경: #E5E7EB - Light gray darker

타이포그래피

  • 탭 버튼 텍스트: 1rem, 중간 굵기
  • 콘텐츠 제목: 1.5rem, 굵게
  • 콘텐츠 본문: 1rem, line-height 1.6

간격

  • 탭 버튼 패딩: 12px 24px
  • 탭 버튼 간격: 8px
  • 콘텐츠 패딩: 32px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="tabs">
  <div class="tab-buttons">
    <button class="tab-btn active" data-tab="0">Tab 1</button>
    <button class="tab-btn" data-tab="1">Tab 2</button>
    <button class="tab-btn" data-tab="2">Tab 3</button>
  </div>
  <div class="tab-contents">
    <div class="tab-panel active">콘텐츠 1</div>
    <div class="tab-panel">콘텐츠 2</div>
    <div class="tab-panel">콘텐츠 3</div>
  </div>
</div>

2단계: 탭 버튼 스타일

.tab-btn {
  padding: 12px 24px;
  background: #F3F4F6;
  border: none;
  border-bottom: 3px solid transparent;
  cursor: pointer;
  transition: 0.3s;
}
.tab-btn.active {
  border-bottom-color: #4F46E5;
}

3단계: JavaScript 탭 전환

const tabBtns = document.querySelectorAll('.tab-btn');
const tabPanels = document.querySelectorAll('.tab-panel');

tabBtns.forEach((btn, index) => {
  btn.addEventListener('click', () => {
    // 모든 active 제거
    tabBtns.forEach(b => b.classList.remove('active'));
    tabPanels.forEach(p => p.classList.remove('active'));

    // 선택된 탭 활성화
    btn.classList.add('active');
    tabPanels[index].classList.add('active');
  });
});
