🎯 주요 학습 포인트

  • Masonry 레이아웃 (Pinterest 스타일)
  • CSS Grid의 grid-auto-flow: dense
  • grid-row: span으로 높이 다르게
  • 불규칙한 카드 배치

📐 퍼블리싱 요구사항

HTML 구조

  • Container <div class="masonry">
  • 12개의 카드, 각각 다른 높이 클래스
  • 카드에 이미지 + 텍스트

CSS 레이아웃

  • Container: display: grid; grid-template-columns: repeat(3, 1fr);
  • Auto rows: grid-auto-rows: 100px;
  • 카드 높이: grid-row: span 2/3/4로 다양하게
  • Gap: 16px

스타일링 세부사항

  • 카드: 배경 흰색, 둥근 모서리, 그림자
  • 이미지: object-fit: cover, 전체 너비
  • 카드 클래스: .tall (span 3), .medium (span 2), .short (span 1)
  • Grid auto-flow: dense로 빈 공간 채우기

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 아티클 카드 (.article-card): 위로 이동, 그림자 강화
  • 더보기 링크 (.read-more): 색상 변경, 화살표 이동

💻 구현 코드

.article-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 20px rgba(0,0,0,0.12);
}
.read-more:hover {
  color: var(--color-primary);
}
.read-more:hover::after {
  transform: translateX(5px);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Container 배경: #F3F4F6 - Light Gray
  • 카드 배경: #FFFFFF - White

타이포그래피

  • 카드 제목: 1rem, 굵게
  • 카드 설명: 0.875rem, #666

간격

  • Grid gap: 16px
  • Auto rows: 100px
  • 카드 패딩: 16px

💡 단계별 구현 가이드

1단계: Masonry Grid

.masonry {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 100px;
  grid-auto-flow: dense;
  gap: 16px;
  padding: 24px;
  background: #F3F4F6;
}

2단계: 카드 높이 변형

.card {
  background: white;
  border-radius: 12px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  overflow: hidden;
}
.card.short { grid-row: span 2; }
.card.medium { grid-row: span 3; }
.card.tall { grid-row: span 4; }

3단계: 카드 내용

.card img {
  width: 100%;
  height: 60%;
  object-fit: cover;
}
.card-content {
  padding: 16px;
}
.card h3 {
  margin-bottom: 8px;
}
.card p {
  font-size: 0.875rem;
  color: #666;
}

✅ 완성 체크리스트

  • □ Grid가 3열로 설정되어 있나요?
  • □ grid-auto-rows가 100px인가요?
  • □ grid-auto-flow: dense가 있나요?
  • □ 카드들이 다른 높이를 가지나요?
  • □ 빈 공간이 자동으로 채워지나요?
  • □ 이미지가 카드에 맞게 조정되나요?
