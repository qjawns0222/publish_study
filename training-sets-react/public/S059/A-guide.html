🎯 주요 학습 포인트

  • 통계 대시보드 카드
  • 숫자와 아이콘 표시
  • 증감률 표시 (▲/▼)
  • Grid 레이아웃

📐 퍼블리싱 요구사항

HTML 구조

  • 4개의 통계 카드
  • 각 카드: 아이콘, 라벨, 숫자, 증감률
  • 증감률에 색상 (증가: 초록, 감소: 빨강)
  • 카드별 다른 아이콘/색상

CSS 레이아웃

  • 그리드: 4열 또는 2x2
  • 아이콘: 큰 아이콘, 배경 원형
  • 숫자: 매우 큰 폰트
  • 증감: 작은 텍스트, 화살표

스타일링 세부사항

  • 아이콘: 각 카드마다 다른 색상
  • 증감률: 양수(▲ 초록), 음수(▼ 빨강)
  • Counter animation (선택사항)
  • Hover 효과

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 통계 카드 (.stat-card): 확대, 그림자 강화

💻 구현 코드

.stat-card {
  transition: 0.3s ease;
}
.stat-card:hover {
  transform: scale(1.03);
  box-shadow: 0 8px 20px rgba(0,0,0,0.12);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • 카드 1 (사용자): #4F46E5 - Primary
  • 카드 2 (매출): #10B981 - Green
  • 카드 3 (주문): #FF9800 - Orange
  • 카드 4 (방문): #3B82F6 - Blue
  • 증가: #10B981 - Green
  • 감소: #EF4444 - Red

타이포그래피

  • 숫자: 2.5rem, 굵게
  • 라벨: 1rem, 회색
  • 증감률: 0.875rem

간격

  • 카드 패딩: 24px
  • 아이콘 크기: 60px
