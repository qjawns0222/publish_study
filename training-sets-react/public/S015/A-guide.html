🎯 주요 학습 포인트

  • Select 드롭다운의 기본 사용법
  • Select 커스텀 스타일링의 한계와 해결 방법
  • <option> 태그로 선택지 구성
  • Select 화살표 커스터마이징 (appearance: none)

📐 퍼블리싱 요구사항

HTML 구조

  • Label + Select 세트 생성
  • Select 안에 5개 이상의 option
  • 첫 번째 option을 placeholder로 사용 (disabled selected)
  • Container로 감싸서 화살표 아이콘 추가

CSS 레이아웃

  • Select: 전체 너비, 패딩 12px, 테두리 스타일
  • Select: appearance: none으로 기본 화살표 제거
  • Container: position: relative
  • 화살표: position: absolute로 우측에 배치

스타일링 세부사항

  • Select 둥근 모서리: 8px
  • Select focus: 테두리 Primary 색상
  • 화살표 아이콘: ▼ 또는 SVG
  • 화살표 위치: right: 12px, 중앙 정렬

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Select 박스 (select): 테두리 강조 (focus)

💻 구현 코드

select:focus {
  border-color: var(--color-primary);
  box-shadow: 0 0 0 3px rgba(79, 70, 229, 0.1);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Select 배경: #FFFFFF - White
  • Select 테두리: #E5E7EB - Light Gray
  • Select focus 테두리: #4F46E5 - Primary
  • 화살표: #666666 - Gray
  • Placeholder: #999999 - Gray

타이포그래피

  • Label: 0.875rem, 굵게
  • Select: 1rem

간격

  • Select 패딩: 12px (좌), 40px (우, 화살표 공간)
  • Select 높이: 48px
  • 화살표 위치: right: 12px
  • Label-Select 간격: 8px

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="form-group">
  <label for="country">국가 선택</label>
  <div class="select-wrapper">
    <select id="country">
      <option value="" disabled selected>국가를 선택하세요</option>
      <option value="kr">대한민국</option>
      <option value="us">미국</option>
      <option value="jp">일본</option>
      <option value="cn">중국</option>
    </select>
    <span class="select-arrow">▼</span>
  </div>
</div>

2단계: Select Wrapper

.select-wrapper {
  position: relative;
  width: 100%;
}

3단계: Select 스타일

select {
  width: 100%;
  height: 48px;
  padding: 12px;
  padding-right: 40px;
  border: 1px solid #E5E7EB;
  border-radius: 8px;
  font-size: 1rem;
  appearance: none;
  background: white;
  cursor: pointer;
  transition: 0.3s ease;
}
select:focus {
  outline: none;
  border-color: var(--color-primary);
}

4단계: 화살표 아이콘

.select-arrow {
  position: absolute;
  right: 12px;
  top: 50%;
  transform: translateY(-50%);
  color: #666;
  pointer-events: none;
  font-size: 0.75rem;
}

✅ 완성 체크리스트

  • □ Select가 appearance: none으로 기본 스타일이 제거되었나요?
  • □ 커스텀 화살표가 우측에 배치되었나요?
  • □ 화살표가 position: absolute로 배치되었나요?
  • □ Select 우측 패딩이 화살표 공간을 확보하나요?
  • □ 첫 번째 option이 placeholder 역할을 하나요?
  • □ Select focus 시 테두리 색상이 변하나요?
  • □ Select 높이가 48px인가요?
  • □ Label과 Select가 올바르게 연결되었나요?
