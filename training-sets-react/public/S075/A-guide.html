🎯 주요 학습 포인트

  • 창의적인 네비게이션 패턴
  • 원형 메뉴 (Circular Menu)
  • transform: rotate로 배치
  • JavaScript로 메뉴 토글

📐 퍼블리싱 요구사항

HTML 구조

  • 중앙 메인 버튼 (+)
  • 5-6개의 서브 메뉴 아이콘
  • 각 메뉴 아이템에 아이콘과 라벨

CSS 레이아웃

  • 메인 버튼: 원형, 중앙 배치
  • 서브 메뉴: position: absolute, transform
  • 애니메이션: 확장/축소, rotate
  • transition: transform 0.3s ease

스타일링 세부사항

  • 메인 버튼 클릭 시 서브 메뉴 펼침
  • 서브 메뉴: 원형으로 배치 (360도 / 개수)
  • transform: rotate + translateY로 위치 계산
  • 각 메뉴 순차적 애니메이션 (transition-delay)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 네비게이션 아이템 (.emotion-nav-item): 확대, 회전, 색상 변경

💻 구현 코드

.emotion-nav-item:hover {
  transform: scale(1.2) rotate(5deg);
  color: var(--color-accent);
  background: rgba(255, 152, 0, 0.1);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

  • 서브 버튼: #10B981 - Green

타이포그래피

  • 아이콘: 1.5rem, 흰색
  • 라벨: 0.75rem, 회색

간격

  • undefined: 50px
  • undefined: 120px

3단계: JavaScript - Toggle

const menuToggle = document.getElementById('menuToggle');
const menuItems = document.getElementById('menuItems');

menuToggle.addEventListener('click', () => {
  menuToggle.classList.toggle('active');
  menuItems.classList.toggle('active');
});

// 외부 클릭 시 메뉴 닫기
document.addEventListener('click', (e) => {
  if (!e.target.closest('.circular-menu')) {
    menuToggle.classList.remove('active');
    menuItems.classList.remove('active');
  }
});
