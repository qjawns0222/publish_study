🎯 주요 학습 포인트

  • 슬라이드 애니메이션으로 메뉴 표시
  • transform: translateX로 좌우 이동
  • JavaScript 없이 CSS만으로 토글 효과 (체크박스 트릭)
  • :checked 선택자 활용

📐 퍼블리싱 요구사항

HTML 구조

  • 토글 버튼 (체크박스 + 라벨)
  • 사이드 메뉴 <div class="sidebar">
  • 메뉴 내부에 링크 5개

CSS 레이아웃

  • Sidebar: 고정 위치 (position: fixed), 너비 250px
  • 초기: transform: translateX(-100%) (화면 밖)
  • 체크박스 checked 시: transform: translateX(0)
  • Transition: transition: transform 0.3s ease

스타일링 세부사항

  • Sidebar: 배경 Dark, 텍스트 흰색, 높이 100vh
  • 체크박스 숨김 처리
  • 토글 버튼: 고정 위치 (top: 20px, left: 20px)

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • 메뉴 토글 버튼 (.menu-toggle): 배경색 변경
  • 메뉴 항목 (.menu-item): 배경색 변경 (hover)

💻 구현 코드

.menu-toggle:hover {
  background: var(--color-primary);
}
.menu-item:hover {
  background: rgba(79, 70, 229, 0.1);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Sidebar 배경: #222222 - Dark
  • Sidebar 링크: #FFFFFF - White
  • 토글 버튼 배경: #4F46E5 - Primary

타이포그래피

  • Sidebar 링크: 1rem

간격

  • Sidebar 너비: 250px
  • Sidebar 패딩: 24px
  • 링크 간격: 16px

💡 단계별 구현 가이드

1단계: HTML 구조

<input type="checkbox" id="menu-toggle" hidden>
<label for="menu-toggle" class="toggle-btn">☰</label>
<div class="sidebar">
  <a href="#">홈</a>
  <a href="#">소개</a>
  <a href="#">서비스</a>
  <a href="#">블로그</a>
  <a href="#">연락처</a>
</div>

2단계: Sidebar 기본

.sidebar {
  position: fixed;
  top: 0;
  left: 0;
  width: 250px;
  height: 100vh;
  background: #222;
  padding: 24px;
  transform: translateX(-100%);
  transition: transform 0.3s ease;
}

3단계: 토글 기능

#menu-toggle:checked ~ .sidebar {
  transform: translateX(0);
}
.toggle-btn {
  position: fixed;
  top: 20px;
  left: 20px;
  padding: 12px 16px;
  background: var(--color-primary);
  color: white;
  border-radius: 8px;
  cursor: pointer;
  z-index: 1000;
}

  • □ Sidebar 높이가 100vh인가요?
