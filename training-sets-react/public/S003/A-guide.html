🎯 주요 학습 포인트

  • CSS Grid를 사용한 2단 컬럼 레이아웃
  • grid-template-columns로 Sidebar와 Content 비율 조정
  • Sidebar의 고정 너비 vs Content의 유동 너비
  • <aside> 시맨틱 태그의 올바른 사용

📐 퍼블리싱 요구사항

HTML 구조

  • Container <div> 안에 <aside>와 <main> 배치
  • <aside> 내부에 네비게이션 링크 리스트
  • <main> 내부에 제목과 본문 콘텐츠

CSS 레이아웃

  • Container: display: grid; grid-template-columns: 250px 1fr;
  • Sidebar(aside): 고정 너비 250px, 배경색 #F3F4F6, 패딩 24px
  • Main: 남는 공간 모두 차지 (1fr), 패딩 48px
  • 최소 높이: min-height: 100vh;

스타일링 세부사항

  • Sidebar 링크: 세로 정렬, 각 링크마다 12px 간격
  • Sidebar 링크 hover: 배경색 rgba(79, 70, 229, 0.1), 좌측 바 표시
  • Main 콘텐츠: 최대 너비 800px

🎨 인터랙션 상세 요구사항

⚠️ 이 예제에서 반드시 hover 효과를 적용해야 할 요소:

📌 인터랙션 대상

  • Sidebar 메뉴 링크 (.sidebar a): 색상 변경, 밑줄 없음

💻 구현 코드

.sidebar a {
  transition: 0.3s ease;
}
.sidebar a:hover {
  color: var(--color-accent);
}
💡 팁: 위 요소들에 transition: 0.3s ease를 먼저 설정한 후 hover 효과를 적용하세요!

🎨 스타일 가이드 (이 예제 전용)

색상 팔레트

  • Sidebar 배경: #F3F4F6 - Light Gray
  • Sidebar 링크 hover: rgba(79, 70, 229, 0.1) - Primary 10%
  • Sidebar 좌측 바: #FF9800 - Accent
  • Main 배경: #FFFFFF - White

타이포그래피

  • Sidebar 링크: 1rem, 보통
  • Main 제목: 2rem, 굵게
  • Main 본문: 1rem, line-height: 1.6

간격

  • Sidebar 패딩: 24px
  • Sidebar 링크 간격: 12px (각 링크)
  • Main 패딩: 48px
  • Sidebar 너비: 250px (고정)

💡 단계별 구현 가이드

1단계: HTML 구조

<div class="container">
  <aside class="sidebar">
    <nav>
      <a href="#">메뉴 1</a>
      <a href="#">메뉴 2</a>
      <a href="#">메뉴 3</a>
    </nav>
  </aside>
  <main>
    <h1>메인 콘텐츠</h1>
    <p>내용...</p>
  </main>
</div>

2단계: Grid 레이아웃

.container {
  display: grid;
  grid-template-columns: 250px 1fr;
  min-height: 100vh;
}

3단계: Sidebar 스타일

.sidebar {
  background: #F3F4F6;
  padding: 24px;
}
.sidebar a {
  display: block;
  padding: 12px;
  margin-bottom: 8px;
  color: #222;
  text-decoration: none;
  transition: 0.3s ease;
}
.sidebar a:hover {
  background: rgba(79, 70, 229, 0.1);
  color: var(--color-accent);
  border-left: 3px solid var(--color-accent);
  padding-left: 20px;
}

4단계: Main 스타일

main {
  padding: 48px;
  max-width: 800px;
}

✅ 완성 체크리스트

  • □ Container가 CSS Grid로 설정되어 있나요?
  • □ grid-template-columns: 250px 1fr;를 사용했나요?
  • □ Sidebar가 고정 너비 250px인가요?
  • □ Main이 남는 공간(1fr)을 차지하나요?
  • □ Sidebar 링크가 세로로 정렬되어 있나요?
  • □ Sidebar 링크 hover 시 배경색이 변하나요?
  • □ Sidebar 링크 hover 시 좌측에 accent 색상 바가 나타나나요?
  • □ 전체 높이가 최소 100vh인가요?
